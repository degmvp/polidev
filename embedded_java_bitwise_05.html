Classe 09 — ChecksumXor.java

Esta classe calcula um checksum usando operação XOR.

O XOR possui uma característica poderosa: bits iguais geram 0 e bits diferentes geram 1. 


public class ChecksumXor {

    private int[] data;

    public ChecksumXor(int[] data) {
        this.data = data;
    }

    public int calculateChecksum() {

        int checksum = 0;

        for (int value : data) {
            checksum ^= value;
        }

        return checksum;
    }

    public String toBinary(int value) {

        return String.format("%8s",
                Integer.toBinaryString(value))
                .replace(' ', '0');
    }

    public void showData() {

        System.out.println("=== DATA PACKET ===");

        for (int value : data) {

            System.out.println(
                value + " -> " + toBinary(value)
            );
        }

        int checksum = calculateChecksum();

        System.out.println();
        System.out.println(
            "CHECKSUM = " +
            checksum +
            " -> " +
            toBinary(checksum)
        );
    }

    public static void main(String[] args) {

        int[] packet = {12, 45, 78, 23, 99};

        ChecksumXor check =
            new ChecksumXor(packet);

        check.showData();
    }
}

Classe 10 — ShiftEngine.java

Esta classe demonstra deslocamento binário para esquerda e direita.

Shift é uma das operações mais importantes da computação. 


public class ShiftEngine {

    private int value;

    public ShiftEngine(int value) {
        this.value = value;
    }

    public int shiftLeft(int bits) {
        return value << bits;
    }

    public int shiftRight(int bits) {
        return value >> bits;
    }

    public String toBinary(int number) {

        return String.format("%16s",
                Integer.toBinaryString(number))
                .replace(' ', '0');
    }

    public void showShifts() {

        System.out.println(
            "ORIGINAL : " +
            value +
            " -> " +
            toBinary(value)
        );

        int left = shiftLeft(2);

        System.out.println(
            "LEFT <<2 : " +
            left +
            " -> " +
            toBinary(left)
        );

        int right = shiftRight(2);

        System.out.println(
            "RIGHT >>2: " +
            right +
            " -> " +
            toBinary(right)
        );
    }

    public static void main(String[] args) {

        ShiftEngine engine =
            new ShiftEngine(25);

        System.out.println(
            "=== SHIFT ENGINE ==="
        );

        engine.showShifts();
    }
}

Saída Esperada — ShiftEngine.java

LEFT <<2 : 100 -> 0000000001100100

RIGHT >>2: 6 -> 0000000000000110

Resumo Técnico


valor ^= outroValor   // XOR

valor << n            // Shift Left
valor >> n            // Shift Right

O operador XOR é muito utilizado em:
