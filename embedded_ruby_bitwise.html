1 - Verificação de Flag Binária

Checa rapidamente se um determinado bit está ativo em uma máscara de status. Muito utilizado em sistemas embarcados e registradores de hardware. 

flags = 0b1010

if (flags & 0b0010) != 0
    puts "Flag ativa"
else
    puts "Flag desligada"
end

2 - Ativando Bits (SET)

Liga um bit específico preservando os demais estados do registrador. 

flags = 0b0000

flags |= 0b0100

puts flags.to_s(2)

3 - Limpando Bits (CLEAR)

Remove apenas o bit desejado sem afetar os demais. 

flags = 0b1111

flags &= ~0b0100

puts flags.to_s(2)

4 - Toggle de Bits

Inverte um bit específico usando XOR. Muito útil para alternar estados ON/OFF. 

flags = 0b1000

flags ^= 0b1000

puts flags.to_s(2)

5 - Simulação de Registrador Embedded

Exemplo simples simulando um registrador de controle. 

register = 0x00

# Liga motor
register |= 0b00000001

# Liga sensor
register |= 0b00000100

puts "Estado: #{register.to_s(2)}"

6 - Shift Left para Multiplicação Rápida

Operações de deslocamento continuam extremamente rápidas e úteis. 

valor = 8

resultado = valor << 2

puts resultado

7 - Shift Right para Divisão Binária

Deslocamento à direita simulando divisão inteira por potência de dois. 

valor = 64

resultado = valor >> 3

puts resultado

8 - Checksum XOR

Técnica clássica de validação leve utilizada em protocolos simples. 

dados = [12, 44, 98, 10]

checksum = 0

dados.each do |b|
    checksum ^= b
end

puts checksum

9 - Compactação de Estados Binários

Empacota múltiplos estados booleanos em apenas um inteiro. 

estado = 0

wifi   = true
gps    = true
sensor = false

estado |= 1 << 0 if wifi
estado |= 1 << 1 if gps
estado |= 1 << 2 if sensor

puts estado.to_s(2)

10 - Extração de Campo Empacotado

Extrai partes específicas de um pacote binário. 

pacote = 0b11010110

campo = (pacote >> 2) & 0b111

puts campo
