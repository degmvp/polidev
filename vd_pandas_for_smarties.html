2. DataFrame Creation

import pandas as pd

dados = {
    "curso": ["Python", "SQL", "Pandas"],
    "downloads": [120, 95, 180]
}

df = pd.DataFrame(dados)

print(df)

3. Filtering Smart Data

top = df[df["downloads"] >= 100]

print(top)

Filtering transforms raw information into real insight.

4. Sorting the Ranking

ranking = df.sort_values(
    by="downloads",
    ascending=False
)

print(ranking)
