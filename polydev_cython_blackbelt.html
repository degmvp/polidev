Brief — O que é Cython?

Cython é uma linguagem de programação que combina Python com elementos de C. Ela permite escrever código com aparência próxima ao Python, mas que pode ser compilado para código de máquina altamente eficiente, semelhante ao C.

Seu principal objetivo é otimizar partes críticas de programas Python, principalmente em cálculos intensivos, manipulação de grandes volumes de dados, processamento científico, machine learning e integração com bibliotecas C/C++.

Resumindo: Cython é uma ponte entre Python e C, permitindo desenvolver código mais rápido e eficiente sem abandonar o ecossistema Python.

2. Filtro de Média Móvel

Filtra dados para suavizar sinais, comum em séries temporais.

def moving_average(np.ndarray[np.float64_t, ndim=1] data,
                   int window_size):
    cdef int n = data.shape[0]
    cdef np.ndarray[np.float64_t, ndim=1] result = np.zeros(n - window_size + 1)
    cdef double soma = 0
    cdef int i

    for i in range(window_size):
        soma += data[i]

    result[0] = soma / window_size

    for i in range(1, n - window_size + 1):
        soma += data[i + window_size - 1] - data[i - 1]
        result[i] = soma / window_size

    return result

5. Multiplicação de Matrizes

Fundamental em álgebra linear, computação científica e aprendizado de máquina.

def matmul(np.ndarray[np.float64_t, ndim=2] A,
           np.ndarray[np.float64_t, ndim=2] B):
    cdef int i, j, k
    cdef int m = A.shape[0]
    cdef int n = B.shape[1]
    cdef int p = A.shape[1]
    cdef np.ndarray[np.float64_t, ndim=2] C = np.zeros((m, n))

    for i in range(m):
        for j in range(n):
            for k in range(p):
                C[i, j] += A[i, k] * B[k, j]

    return C

8. Variância de Dados

Calcula a dispersão estatística de um conjunto de dados.

def variance(np.ndarray[np.float64_t, ndim=1] data):
    cdef double mean = 0
    cdef double var = 0
    cdef int n = data.shape[0]
    cdef int i

    for i in range(n):
        mean += data[i]

    mean /= n

    for i in range(n):
        var += (data[i] - mean) ** 2

    return var / n

9. Interpolação Linear

Estima valores intermediários entre dois pontos conhecidos.

def linear_interpolation(double x0, double y0,
                         double x1, double y1,
                         double x):
    return y0 + (y1 - y0) * (x - x0) / (x1 - x0)
