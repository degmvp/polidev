O que é Pandas?

Pandas é a principal biblioteca Python para manipulação e análise de dados. Ela permite trabalhar com tabelas, arquivos CSV, Excel, estatísticas, agrupamentos, filtros e preparação de dados para IA e Machine Learning.

  • ✔ Data Science
  • ✔ Business Intelligence
  • ✔ ETL
  • ✔ Dashboards
  • ✔ Streamlit
  • ✔ Machine Learning

Importando Pandas

O padrão mundial é utilizar o alias pd. 

import pandas as pd

Criando um DataFrame

import pandas as pd

dados = {
    "nome": ["Ana", "Carlos", "Maria"],
    "idade": [25, 30, 28]
}

df = pd.DataFrame(dados)

print(df)

Pandas + Streamlit

A combinação Pandas + Streamlit permite construir dashboards, painéis analíticos e aplicações web extremamente rápidas. 

import streamlit as st
import pandas as pd

df = pd.read_csv("dados.csv")

st.dataframe(df)
