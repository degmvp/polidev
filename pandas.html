O que é Pandas?

Pandas é uma biblioteca Python usada para manipulação, análise, limpeza e transformação de dados. É uma ferramenta essencial para Data Science, BI, ETL, Machine Learning e dashboards.

Onde o Pandas é usado?

  • Leitura e gravação de arquivos CSV
  • Leitura e gravação de planilhas Excel
  • Criação de DataFrames
  • Filtros, ordenações e agrupamentos
  • Tratamento de valores nulos
  • Preparação de dados para IA e Machine Learning
  • Integração com Streamlit para dashboards

Importação padrão

O alias mundialmente utilizado para Pandas é pd. 

import pandas as pd

Exemplo básico

import pandas as pd

dados = {
    "nome": ["Ana", "Carlos", "Maria"],
    "idade": [25, 30, 28]
}

df = pd.DataFrame(dados)

print(df)
