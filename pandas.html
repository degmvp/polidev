O que é Pandas?

Pandas é uma das bibliotecas mais importantes do ecossistema Python. Ela permite manipular tabelas, arquivos CSV, Excel, análises estatísticas, tratamento de dados e preparação para Machine Learning.

O Pandas é amplamente utilizado em:

  • Data Science
  • Business Intelligence
  • ETL
  • Machine Learning
  • Análise Financeira
  • Dashboards
  • Streamlit

Importando Pandas

import pandas as pd

O alias padrão mundial utilizado é pd.

Exemplo de DataFrame

import pandas as pd

dados = {
    "nome": ["Ana", "Carlos", "Maria"],
    "idade": [25, 30, 28]
}

df = pd.DataFrame(dados)

print(df)
