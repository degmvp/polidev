Artigo 03 de 05 — Toggle Binário e Dados Empacotados

Visão Geral

Neste terceiro artigo da série Embedded Java Bitwise, vamos trabalhar com duas classes muito importantes:

Aqui entramos em uma área muito usada em protocolos, compactação, drivers, engines e sistemas que precisam economizar memória.

Classe 05 — BinaryToggle.java

Esta classe demonstra como alternar bits usando XOR. Quando o bit está desligado, ele liga. Quando está ligado, ele desliga. 

public class BinaryToggle {

    private static final int LED_RED   = 1;   // 0001
    private static final int LED_GREEN = 2;   // 0010
    private static final int LED_BLUE  = 4;   // 0100
    private static final int BUZZER    = 8;   // 1000

    private int deviceState;

    public BinaryToggle() {
        this.deviceState = 0;
    }

    public void toggle(int flag) {
        deviceState ^= flag;
    }

    public boolean isOn(int flag) {
        return (deviceState & flag) != 0;
    }

    public void showBinary() {
        System.out.println(
            "STATE: " +
            String.format("%4s", Integer.toBinaryString(deviceState))
            .replace(' ', '0')
        );
    }

    public void showStatus() {
        System.out.println("LED_RED   : " + isOn(LED_RED));
        System.out.println("LED_GREEN : " + isOn(LED_GREEN));
        System.out.println("LED_BLUE  : " + isOn(LED_BLUE));
        System.out.println("BUZZER    : " + isOn(BUZZER));
    }

    public static void main(String[] args) {

        BinaryToggle panel = new BinaryToggle();

        System.out.println("=== FIRST TOGGLE ===");

        panel.toggle(LED_RED);
        panel.toggle(LED_BLUE);

        panel.showBinary();
        panel.showStatus();

        System.out.println();

        System.out.println("=== SECOND TOGGLE ===");

        panel.toggle(LED_RED);
        panel.toggle(BUZZER);

        panel.showBinary();
        panel.showStatus();
    }
}

Classe 06 — PackedData.java

Esta classe mostra como empacotar vários campos pequenos dentro de um único inteiro usando deslocamento e máscaras binárias. 

public class PackedData {

    // Layout de bits:
    // bits 0-3   = sensorId   (0 a 15)
    // bits 4-7   = statusCode (0 a 15)
    // bits 8-15  = value      (0 a 255)

    private int data;

    public PackedData() {
        this.data = 0;
    }

    public void pack(int sensorId, int statusCode, int value) {

        data = 0;

        data |= (sensorId & 0x0F);
        data |= (statusCode & 0x0F) << 4;
        data |= (value & 0xFF) << 8;
    }

    public int getSensorId() {
        return data & 0x0F;
    }

    public int getStatusCode() {
        return (data >> 4) & 0x0F;
    }

    public int getValue() {
        return (data >> 8) & 0xFF;
    }

    public void showBinary() {
        System.out.println(
            "PACKED: " +
            String.format("%16s", Integer.toBinaryString(data))
            .replace(' ', '0')
        );
    }

    public void showData() {
        System.out.println("Sensor ID   : " + getSensorId());
        System.out.println("Status Code : " + getStatusCode());
        System.out.println("Value       : " + getValue());
    }

    public static void main(String[] args) {

        PackedData packet = new PackedData();

        System.out.println("=== PACKING DATA ===");

        packet.pack(9, 3, 120);

        packet.showBinary();
        packet.showData();
    }
}

=== PACKING DATA ===

Sensor ID   : 9
Status Code : 3
Value       : 120

Resumo Técnico

valor ^= flag;          // Alterna um bit

data |= campo;          // Grava campo
data |= campo << n;     // Desloca campo para posição correta

data & mascara;         // Extrai campo
(data >> n) & mascara;  // Desloca e extrai  campo

O operador XOR é perfeito para alternar estados. Já o empacotamento binário permite guardar vários valores pequenos dentro de uma única variável.
