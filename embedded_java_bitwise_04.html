Classe 07 — BitCounter.java

Esta classe conta quantos bits estão ativos em um número inteiro. Essa técnica também é conhecida como popcount. 

public class BitCounter {

    private int value;

    public BitCounter(int value) {
        this.value = value;
    }

    public int countBitsSimple() {

        int count = 0;
        int temp = value;

        while (temp > 0) {

            if ((temp & 1) == 1) {
                count++;
            }

            temp >>= 1;
        }

        return count;
    }

    public int countBitsFast() {

        int count = 0;
        int temp = value;

        while (temp != 0) {

            temp &= (temp - 1);
            count++;
        }

        return count;
    }

    public void showBinary() {
        System.out.println(
            "VALUE: " +
            String.format("%8s", Integer.toBinaryString(value))
            .replace(' ', '0')
        );
    }

    public void showResult() {
        System.out.println("Simple Count : " + countBitsSimple());
        System.out.println("Fast Count   : " + countBitsFast());
    }

    public static void main(String[] args) {

        BitCounter counter = new BitCounter(173);

        System.out.println("=== BIT COUNTER ===");

        counter.showBinary();
        counter.showResult();
    }
}

Classe 08 — NetworkMask.java

Esta classe simula o cálculo de endereço de rede IPv4 usando operação AND entre o IP e a máscara de rede. 

public class NetworkMask {

    private int ip;
    private int mask;

    public NetworkMask(int ip, int mask) {
        this.ip = ip;
        this.mask = mask;
    }

    public int getNetworkAddress() {
        return ip & mask;
    }

    public String toBinary(int value) {
        return String.format("%32s", Integer.toBinaryString(value))
                .replace(' ', '0');
    }

    public String toIPv4(int value) {

        int b1 = (value >> 24) & 0xFF;
        int b2 = (value >> 16) & 0xFF;
        int b3 = (value >> 8) & 0xFF;
        int b4 = value & 0xFF;

        return b1 + "." + b2 + "." + b3 + "." + b4;
    }

    public void showNetwork() {

        int network = getNetworkAddress();

        System.out.println("IP      : " + toIPv4(ip));
        System.out.println("MASK    : " + toIPv4(mask));
        System.out.println("NETWORK : " + toIPv4(network));

        System.out.println();

        System.out.println("IP BIN      : " + toBinary(ip));
        System.out.println("MASK BIN    : " + toBinary(mask));
        System.out.println("NETWORK BIN : " + toBinary(network));
    }

    public static void main(String[] args) {

        int ip   = (192 << 24) | (168 << 16) | (10 << 8) | 55;
        int mask = (255 << 24) | (255 << 16) | (255 << 8) | 0;

        NetworkMask net = new NetworkMask(ip, mask);

        System.out.println("=== NETWORK MASK ===");

        net.showNetwork();
    }
}

Resumo Técnico

valor & 1              // Testa o bit menos significativo
valor >>= 1            // Desloca bits para direita
valor &= (valor - 1)   // Remove o bit ativo mais à direita

ip & mask              // Calcula endereço de  rede
(value >> n) & 0xFF    // Extrai octeto IPv4

A contagem de bits ativos é usada em otimizações, compactação, criptografia, inteligência artificial, bancos de dados e algoritmos.

O cálculo de rede por máscara mostra uma aplicação clássica e poderosa da operação AND em sistemas reais.
