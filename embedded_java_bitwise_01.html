Artigo 01 de 05 — Classes Java para Devs que Pensam em Bits

Visão Geral

Neste primeiro artigo da série Embedded Java Bitwise, vamos trabalhar com duas classes fundamentais:

  • BitFlag.java — controle de permissões com máscara binária
  • DeviceRegister.java — simulação de registrador embarcado

A proposta é mostrar Java com classe, encapsulamento, métodos e lógica bitwise aplicada em cenários reais.

Classe 01 — BitFlag.java

Esta classe controla permissões usando bits individuais dentro de um único inteiro. 

public class BitFlag {

    private static final int READ    = 1;   // 0001
    private static final int WRITE   = 2;   // 0010
    private static final int EXECUTE = 4;   // 0100
    private static final int DELETE  = 8;   // 1000

    private int permissions;

    public BitFlag() {
        this.permissions = 0;
    }

    public void addPermission(int permission) {
        permissions |= permission;
    }

    public void removePermission(int permission) {
        permissions &= ~permission;
    }

    public boolean hasPermission(int permission) {
        return (permissions & permission) != 0;
    }

    public void showBinary() {
        System.out.println(
            "BINÁRIO : " +
            String.format("%4s", Integer.toBinaryString(permissions))
            .replace(' ', '0')
        );
    }

    public void showPermissions() {
        System.out.println("READ    : " + hasPermission(READ));
        System.out.println("WRITE   : " + hasPermission(WRITE));
        System.out.println("EXECUTE : " + hasPermission(EXECUTE));
        System.out.println("DELETE  : " + hasPermission(DELETE));
    }

    public static void main(String[] args) {

        BitFlag user = new BitFlag();

        System.out.println("=== ADICIONANDO FLAGS ===");

        user.addPermission(READ);
        user.addPermission(WRITE);
        user.addPermission(EXECUTE);

        user.showBinary();
        user.showPermissions();

        System.out.println();

        System.out.println("=== REMOVENDO WRITE ===");

        user.removePermission(WRITE);

        user.showBinary();
        user.showPermissions();
    }
}

Classe 02 — DeviceRegister.java

Esta classe simula um registrador de dispositivo embarcado, onde cada bit representa um estado do hardware. 

public class DeviceRegister {

    private static final int POWER_ON  = 1;   // 0001
    private static final int READY     = 2;   // 0010
    private static final int ERROR     = 4;   // 0100
    private static final int INTERRUPT = 8;   // 1000

    private int register;

    public DeviceRegister() {
        this.register = 0;
    }

    public void setBit(int bit) {
        register |= bit;
    }

    public void clearBit(int bit) {
        register &= ~bit;
    }

    public boolean isSet(int bit) {
        return (register & bit) != 0;
    }

    public void printRegister() {
        System.out.println(
            "REGISTER: " +
            String.format("%4s", Integer.toBinaryString(register))
            .replace(' ', '0')
        );
    }

    public void showStatus() {
        System.out.println("POWER_ON  : " + isSet(POWER_ON));
        System.out.println("READY     : " + isSet(READY));
        System.out.println("ERROR     : " + isSet(ERROR));
        System.out.println("INTERRUPT : " + isSet(INTERRUPT));
    }

    public static void main(String[] args) {

        DeviceRegister device = new DeviceRegister();

        System.out.println("=== DEVICE START ===");

        device.setBit(POWER_ON);
        device.setBit(READY);

        device.printRegister();
        device.showStatus();

        System.out.println();

        System.out.println("=== ERROR INTERRUPT ===");

        device.setBit(ERROR);
        device.setBit(INTERRUPT);
        device.clearBit(READY);

        device.printRegister();
        device.showStatus();
    }
}

ERROR     : false
INTERRUPT : false

=== ERROR INTERRUPT ===

READY     : false
ERROR     : true
INTERRUPT : true

Resumo Técnico

valor |= bit;      // Liga um bit
valor &= ~bit;     // Desliga um bit
(valor & bit) != 0 // Testa se o bit está ativo

Essas três operações formam a base de controle binário em permissões, registradores, sensores, flags, protocolos e sistemas embarcados.
