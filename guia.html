Missão: Criar uma biblioteca de conteúdos técnicos operando como sistema em nuvem

📌 VISÃO GERAL

O POLYDEV é uma plataforma web que funciona como uma biblioteca de conteúdos técnicos, permitindo armazenar, gerenciar e distribuir arquivos através de um sistema integrado com banco de dados e armazenamento em nuvem.

⚙️ ARQUITETURA DO SISTEMA

O sistema é dividido em três camadas principais:

  • Frontend: HTML + CSS + JavaScript
  • Backend (BaaS): Supabase
  • Storage: Supabase Storage (arquivos)

🔄 FLUXO OPERACIONAL

1. Upload de Conteúdo

Admin envia arquivo → Storage salva → URL gerada → registro no banco

2. Listagem

Sistema consulta tabela "conteudos" → dados exibidos

3. Download

Clique → registra download → abre arquivo

4. Edição

Altera dados → atualiza banco → reflete na tela

5. Exclusão

Verifica FK → impede exclusão com histórico
