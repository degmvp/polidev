🧠 Situação Atual do Projeto

O projeto POLYDEV evoluiu de uma ideia educacional para uma aplicação funcional baseada em arquitetura moderna e leve. Hoje o sistema opera sem backend dedicado, utilizando serviços cloud e integração direta via API.

Arquitetura implementada

  • Frontend: GitHub Pages (HTML + JavaScript)
  • API: Supabase REST
  • Banco de Dados: PostgreSQL
  • Administração: DBeaver
  • Modelo: Serverless
---

⚙️ Funcionalidades já implementadas

  • Cadastro de usuários funcional
  • Integração completa com banco de dados
  • Validação de formulário
  • Registro de downloads
  • Estrutura de conteúdos (scripts e materiais)
  • Dashboard dinâmico com métricas reais
---

📊 Sistema Analítico (Destaque)

Um dos pontos mais importantes do projeto foi a criação de um sistema de análise de dados interno.

  • Tabela analítica: dashgeral
  • Atualização automática via trigger
  • Agregação por categoria, conteúdo e usuário
  • Leitura simplificada via API REST

Isso transformou o projeto em algo além de um site — tornou-se uma pequena plataforma de dados.

---

🚀 Avaliação Técnica

  • Arquitetura: ✔ Moderna (Serverless)
  • Banco de dados: ✔ Bem estruturado
  • Integração API: ✔ Funcional
  • Automação (Trigger): ✔ Profissional
