📌 O que é Streamlit?

Streamlit é uma biblioteca Python moderna utilizada para criação rápida de aplicações web interativas, dashboards, ferramentas de dados, aplicações de IA e sistemas visuais utilizando apenas Python.

Com poucas linhas de código é possível criar:

  • Dashboards
  • Sistemas administrativos
  • Aplicações IA
  • Visualização de dados
  • Interfaces para Machine Learning
  • Ferramentas analíticas
  • Aplicações web locais

⚙ Instalação

pip install streamlit

Após a instalação: 

streamlit run app.py

🧠 Primeiro Programa

import streamlit as st

st.title("POLYDEV Streamlit")

st.write("Olá mundo Streamlit!")
