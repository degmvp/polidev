1 - Batch Processing para Grandes Volumes

Processamento em lotes reduz drasticamente o consumo de memória. 

def batch_process_large_array(data, batch_size=10000):

    results = []

    for i in range(0, len(data), batch_size):

        batch = data[i:i+batch_size]

        batch_norm = (
            batch - batch.mean(axis=0)
        ) / (
            batch.std(axis=0) + 1e-8
        )

        results.append(batch_norm)

    return np.vstack(results)

Excelente para pipelines de machine learning e datasets gigantescos.

2 - Lookup Ultra-Rápido

Uso de searchsorted para buscas extremamente eficientes. 

def fast_lookup(codes, values, lookup_table):

    sorted_idx = np.argsort(codes)

    sorted_codes = codes[sorted_idx]

    positions = np.searchsorted(sorted_codes, values)

    return lookup_table[sorted_idx[positions]]

Muito mais rápido que loops tradicionais em Python.

3 - Filtragem Vetorizada

def filter_outliers(data, n_std=3.0):

    mean = np.mean(data)

    std = np.std(data)

    mask = np.abs(data - mean) < n_std * std

    return data[mask], mask

Remoção de outliers usando operações totalmente vetorizadas.

4 - Rolling Window Sem Cópia

def rolling_window(a, window):

    shape = a.shape[:-1] + (
        a.shape[-1] - window + 1,
        window
    )

    strides = a.strides + (
        a.strides[-1],
    )

    return np.lib.stride_tricks.as_strided(
        a,
        shape=shape,
        strides=strides
    )

Técnica extremamente poderosa para análise temporal e séries financeiras.

5 - Group Normalization

def group_normalize(data, groups):

    unique_groups = np.unique(groups)

    result = np.zeros_like(data, dtype=float)

    for g in unique_groups:

        mask = groups == g

        group_data = data[mask]

        result[mask] = (
            group_data - group_data.mean()
        ) / (
            group_data.std() + 1e-8
        )

    return result

Muito utilizado em analytics, deep learning e pipelines estatísticos.

6 - Pairwise Distance com Broadcasting

def pairwise_distance(X, Y=None):

    if Y is None:
        Y = X

    X2 = np.sum(X**2, axis=1, keepdims=True)

    Y2 = np.sum(Y**2, axis=1)

    XY = X @ Y.T

    return np.sqrt(X2 + Y2 - 2 * XY)

Fundamental em clustering, KNN e algoritmos geométricos.

def fast_linear_interpolate(x, y, x_new):

Solução extremamente rápida para interpolação numérica.

8 - Fast Smooth

def fast_smooth(data, window_size=5):

    kernel = np.ones(window_size) / window_size

    return np.convolve(
        data,
        kernel,
        mode='valid'
    )

Muito utilizado em:

  • DSP
  • Séries temporais
  • Suavização estatística
  • Pré-processamento ML

9 - Top-K Ultra-Eficiente

def top_k_fast(arr, k=10):

    idx = np.argpartition(arr, -k)[-k:]

    idx = idx[np.argsort(arr[idx])[::-1]]

    return arr[idx], idx

Muito mais eficiente que argsort completo.

10 - Sparse Correlation

def sparse_correlation(a, b, threshold=0.01):

    mask = (
        np.abs(a) > threshold
    ) | (
        np.abs(b) > threshold
    )

    if mask.sum() == 0:
        return 0.0

    return np.corrcoef(
        a[mask],
        b[mask]
    )[0,1]

Excelente para dados esparsos e otimização de memória.
