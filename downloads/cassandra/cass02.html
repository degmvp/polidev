Apache Cassandra – LAB 02
Partition Key & Clustering Key

Neste laboratório você entenderá como o Cassandra distribui e organiza dados utilizando Partition Key e Clustering Key.

Objetivo do LAB
  • Entender o papel da Partition Key
  • Entender o papel da Clustering Key
  • Criar tabelas corretamente modeladas
  • Executar consultas válidas em Cassandra

1️⃣ Conceito de Partition Key

A Partition Key define:

  • Em qual nó os dados serão armazenados
  • Como os dados são distribuídos no cluster
Regra de Ouro
A maioria das consultas em Cassandra deve usar a Partition Key.

2️⃣ Criando tabela com Partition Key

CREATE TABLE clientes_por_cidade (
    cidade text,
    id UUID,
    nome text,
    email text,
    PRIMARY KEY (cidade, id)
);
Explicação:
  • cidade → Partition Key
  • id → Clustering Key

3️⃣ Inserindo Dados

INSERT INTO clientes_por_cidade (cidade, id, nome, email)
VALUES ('Recife', uuid(), 'Ana Silva', 'ana@email.com');

INSERT INTO clientes_por_cidade (cidade, id, nome, email)
VALUES ('Recife', uuid(), 'João Costa', 'joao@email.com');

INSERT INTO clientes_por_cidade (cidade, id, nome, email)
VALUES ('São Paulo', uuid(), 'Carlos Lima', 'carlos@email.com');

4️⃣ Consultando com Partition Key

SELECT * FROM clientes_por_cidade
WHERE cidade = 'Recife';
✔ Consulta válida
Utiliza a Partition Key.

5️⃣ Consulta INVÁLIDA

SELECT * FROM clientes_por_cidade
WHERE nome = 'Ana Silva';
Erro esperado:
Cassandra não permite consultas sem a Partition Key.

6️⃣ Clustering Key

A Clustering Key define a ordem dos dados dentro da mesma partição. 

CREATE TABLE acessos_por_usuario (
    usuario text,
    data_acesso timestamp,
    ip text,
    PRIMARY KEY (usuario, data_acesso)
) WITH CLUSTERING ORDER BY (data_acesso DESC);

7️⃣ Consulta usando Clustering Key

SELECT * FROM acessos_por_usuario
WHERE usuario = 'ana'
LIMIT 5;
Benefício
Busca rápida dos últimos registros por usuário.

8️⃣ Boas Práticas

  • Modele a tabela conforme a consulta
  • Evite tabelas genéricas
  • Duplicar dados é normal em Cassandra
  • Partition Key mal escolhida gera gargalos
