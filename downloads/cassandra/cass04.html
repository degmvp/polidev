Apache Cassandra – LAB 04
Índices Secundários

Neste laboratório você aprenderá como funcionam os Secondary Indexes, quando utilizá-los e quais erros comuns devem ser evitados.

Objetivo do LAB
  • Criar índices secundários
  • Executar consultas usando índices
  • Entender limitações
  • Analisar erros reais do Cassandra

1️⃣ O que é um Índice Secundário?

Índices secundários permitem consultas por colunas que não fazem parte da chave primária.

Importante
Índices secundários não substituem modelagem correta.

2️⃣ Criando Índice Secundário

Tabela base utilizada: 

CREATE TABLE clientes (
    id UUID PRIMARY KEY,
    nome   text,
    email  text,
    cidade text
);

CREATE INDEX idx_clientes_email
ON clientes (email);

3️⃣ Consultando usando Índice

SELECT * FROM clientes
WHERE email = 'ana@email.com';
✔ Consulta permitida
O índice secundário é utilizado.

4️⃣ Erro Real: INDEX_BUILD_IN_PROGRESS

ReadFailure: Error from server: code=1300 [Replica(s) failed to execute read]
message="Operation failed - received 0 responses and 2 failures:
INDEX_BUILD_IN_PROGRESS"
Causa:
O índice ainda estava sendo construído no cluster.
Solução
  • Aguardar a finalização do build do índice
  • Evitar consultas durante o build

5️⃣ Verificando Índices Existentes

No Cassandra, o comando abaixo não funciona: 

DESCRIBE INDEXES ON clientes;
Comando correto: 
DESCRIBE TABLE clientes;

O índice aparecerá na definição da tabela.

6️⃣ Quando NÃO usar Índice Secundário

  • Tabelas com alto volume de dados
  • Alta cardinalidade
  • Consultas frequentes
  • Ambientes de alta escrita

7️⃣ Alternativa Correta

Em Cassandra, a alternativa ao índice secundário é:

✔ Criar tabela específica para a consulta 
CREATE TABLE clientes_por_email (
    email text PRIMARY KEY,
    id UUID,
    nome text,
    cidade text
);

8️⃣ Boas Práticas

  • Use índice apenas em LAB ou baixa cardinalidade
  • Prefira tabelas modeladas por consulta
  • Evite índices em produção
  • Conheça o padrão de acesso antes de modelar

9️⃣ Encerramento do LAB

  • Você criou índices secundários
  • Entendeu erros comuns
  • Aprendeu alternativas corretas
