1. Objetivo do Guia

Este guia foi criado para servir como material de apoio aos códigos aplicados do sistema POLYDEV. As operações bitwise trabalham diretamente com os bits dos números inteiros, permitindo manipular máscaras, permissões, flags, registradores, compressão, redes, criptografia e estruturas internas de sistemas.

2. Conceito Básico

Todo número inteiro pode ser representado em binário. Por exemplo: 
Decimal 10 = Binário 1010
Decimal 12 = Binário 1100
Decimal 15 = Binário 1111

3. Operadores Bitwise

AND — operador &

Retorna 1 somente quando os dois bits comparados forem 1.

12 = 1100
10 = 1010
---------
8  = 1000

a = 12
b = 10

resultado = a & b
print(resultado)  # 8
OR — operador |

Retorna 1 quando pelo menos um dos bits for 1.

12 = 1100
10 = 1010
---------
14 = 1110

a = 12
b = 10

resultado = a | b
print(resultado)  # 14
XOR — operador ^

Retorna 1 quando os bits forem diferentes.

12 = 1100
10 = 1010
---------
6  = 0110

a = 12
b = 10

resultado = a ^ b
print(resultado)  # 6
NOT — operador ~

Inverte os bits. Em Python, o resultado considera representação com sinal.

a = 5

resultado = ~a
print(resultado)  # -6
Shift Left — operador <<

Move os bits para a esquerda. Na prática, multiplica por potências de 2.

3 = 0011

3 << 1 = 0110 = 6

a = 3
print(a << 1)  # 6
Shift Right — operador >>

Move os bits para a direita. Na prática, divide por potências de 2.

8 = 1000

8 >> 1 = 0100 = 4

a = 8
print(a >> 1)  # 4

4. Exemplos Aplicados

Verificar se número é par

Se o último bit for 0, o número é par.

n = 14

if n & 1 == 0:
    print("Par")
Verificar se número é ímpar

Se o último bit for 1, o número é ímpar.

n = 15

if n & 1:
    print("Ímpar")
Ligar um bit

Usamos OR para ativar um bit específico.

numero = 8      # 1000
mascara = 2     # 0010

resultado = numero | mascara

print(resultado)  # 10
Desligar um bit

Usamos AND combinado com NOT para remover um bit ativo.

numero = 10     # 1010
mascara = 2     # 0010

resultado = numero & ~mascara

print(resultado)  # 8
Alternar um bit

Usamos XOR para trocar o estado do bit.

numero = 10     # 1010
mascara = 2     # 0010

resultado = numero ^ mascara

print(resultado)  # 8
Verificar potência de 2

Potências de 2 possuem apenas um bit ligado.

n = 16

if n & (n - 1) == 0:
    print("Potência de 2")
Contar bits ligados

Conta quantos bits 1 existem em um número.

n = 13  # 1101
contador = 0

while n:
    contador += n & 1
    n >>= 1

print(contador)  # 3

5. Bitwise como Permissões

Um dos usos mais importantes de bitwise é criar permissões usando flags. Cada permissão ocupa uma posição binária. 
READ  = 1   # 0001
WRITE = 2   # 0010
EXEC  = 4   # 0100
ADMIN = 8   # 1000
Criando permissões
READ  = 1
WRITE = 2
EXEC  = 4

usuario = READ | WRITE

print(usuario)  # 3

O usuário recebeu permissão de leitura e escrita.

Testando permissão
READ = 1
usuario = 3

if usuario & READ:
    print("Pode ler")
Removendo permissão
WRITE = 2
usuario = 3

usuario = usuario & ~WRITE

print(usuario)  # 1

6. Mapeamento Geral

Tabela resumida das principais operações: 
AND          &     Compara bits
OR           |     Liga bits
XOR          ^     Alterna bits diferentes
NOT          ~     Inverte bits
SHIFT LEFT   <<    Multiplica por 2
SHIFT RIGHT  >>    Divide por 2

8. Nota de Evolução do Sistema

Durante o top-down da operação do sistema POLYDEV, foi identificado e corrigido um erro de fluxo: a página home estava chamando uma landing page usando o código de index.html, o que não fazia sentido dentro da arquitetura atual.
