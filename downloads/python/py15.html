✅ Introdução

Este script apresenta como utilizar a biblioteca matplotlib para ler e exibir uma imagem no formato .jpg ou .png. Ele é útil para visualizar arquivos gráficos em aplicações científicas ou educacionais.

✅ Importação das Bibliotecas


import matplotlib.pyplot as plt
import matplotlib.cbook as cbook

✅ Caminho e Leitura da Imagem

O caminho do arquivo é recuperado com a função get_sample_data() que localiza o arquivo indicado: 


image_file = cbook.get_sample_data('P:\\app_prog_xpydb\\app_abmzs\\modulo.png')
image = plt.imread(image_file)

✅ Exibição da Imagem


fig, ax = plt.subplots()
ax.imshow(image)
ax.axis('off')  # Oculta os eixos
plt.show()

O método imshow() renderiza a imagem na interface gráfica do matplotlib, enquanto axis('off') oculta os eixos X e Y.
