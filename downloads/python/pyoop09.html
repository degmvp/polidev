Tipos de Classes em Python

Existem diversos tipos de classes que podemos usar para diferentes necessidades no desenvolvimento:

1. Classe Concreta

Classe comum que pode ser instanciada normalmente.

class Pessoa:
def __init__(self, nome):
self.nome = nome

2. Classe Abstrata

Classe que não pode ser instanciada diretamente e serve como base para outras classes.

from abc import ABC, abstractmethod

class Animal(ABC):
@abstractmethod
def emitir_som(self):
pass

3. Classe Estática

Classe que possui apenas métodos estáticos, que podem ser chamados sem criar instâncias.

class Utilidades:
@staticmethod
def somar(a, b):
return a + b

print(Utilidades.somar(2, 3)) # Saída: 5

4. Classe Selada (Final)

Em Python não há suporte nativo para classes seladas (final), mas podemos usar bibliotecas para evitar herança.

5. Classe Parcial

Python não suporta classes parciais nativamente como C#, mas é possível modularizar funcionalidades usando múltiplos arquivos e importações.

Resumo

Conhecer os diferentes tipos de classes ajuda a estruturar melhor o sistema, aproveitando as características de cada tipo para manter código limpo e organizado.
