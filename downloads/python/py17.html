✅ Empilhando Decorators

Dois decoradores declara1 e declara2 são aplicados à função imprimir_nome(). A ordem de execução segue de baixo para cima. 


def declara1(func1):
    def function_decoradora1(name):
        print('função --> declara1 --> chamada como function_decoradora1')
        func1(name)
    return function_decoradora1

def declara2(func2):
    def function_decoradora2(n):
        print('função --> declara2 chamada como function_decoradora2')
        func2(n)
    return function_decoradora2

@declara1
@declara2
def imprimir_nome(name):
    print(name)

imprimir_nome('Apos usar function_decoradora1-2 imprime o nome: Degmar')

✅ Decorator para exibir mensagem com resultado


def p_decorador(func):
    def function_decoradora(n):
        print('Usando Decorator', func(n))
    return function_decoradora

✅ Decorator alternativo com mensagem personalizada


def p_decorador1(func):
    def function_decoradora(n):
        print('Testando Decorator', func(n))
    return function_decoradora

✅ Função Fibonacci com decorador


@p_decorador
def fib_dyn(n):
    if n == 1 or n == 2:
        return 1
    f1 = 1
    f2 = 1
    for i in range(3, n + 1):
        f = f1 + f2
        f2 = f1
        f1 = f
    return f1

print(fib_dyn(17))
print(fib_dyn(27))

✅ Nova versão de Fibonacci com outro decorador


@p_decorador1
def fib_dyn(n):

Decoradores são extremamente úteis para adicionar funcionalidades a funções de forma reutilizável e limpa, sem alterar seu código interno.
