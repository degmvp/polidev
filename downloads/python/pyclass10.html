Módulo 10 - Iterables em Python

Explore os diferentes tipos de objetos iteráveis em Python e suas particularidades.

1. O que é um iterable

Um iterable é qualquer objeto que pode ser percorrido em um loop.

Exercício Prático:

# Objetos iteráveis comuns lista = [1, 2, 3] # Lista tupla = (1, 2, 3) # Tupla string = "Python" # String dicionario = {'a': 1, 'b': 2} # Dicionário (itera nas chaves) conjunto = {1, 2, 3} # Conjunto range_obj = range(5) # Range # Todos podem ser usados em loops for item in lista: print(item) # Verificando se é iterável from collections.abc import Iterable print(isinstance(5, Iterable)) # False print(isinstance("texto", Iterable)) # True
Dica: Qualquer objeto que implemente o método __iter__() é um iterable.

Fixando o Conhecimento:

Teste quais destes objetos são iteráveis:

  1. 123
  2. [1, 2, 3]
  3. {'a': 1, 'b': 2}
  4. True
  5. range(10)

2. Range

range gera uma sequência imutável de números, eficiente em memória.

Exercício Prático:

# Formas de criar range r1 = range(5) # 0, 1, 2, 3, 4 r2 = range(1, 6) # 1, 2, 3, 4, 5 r3 = range(0, 10, 2) # 0, 2, 4, 6, 8 r4 = range(10, 0, -1) # 10, 9, ..., 1 # Convertendo para lista print(list(r3)) # [0, 2, 4, 6, 8] # Usos comuns for i in range(3): print("Python") indices = range(len(["a", "b", "c"])) print(list(indices)) # [0, 1, 2]
Performance: range não cria a lista na memória, gerando os números sob demanda.

Fixando o Conhecimento:

Crie ranges para:

  1. Números de 0 a 9
  2. Números pares de 2 a 20
  3. Contagem regressiva de 5 a 1
  4. Múltiplos de 3 entre 30 e 0

3. Set

Conjuntos são coleções não ordenadas de elementos únicos.

Exercício Prático:

# Criando conjuntos frutas = {"maçã", "banana", "laranja"} numeros = {1, 2, 3, 3, 2} # {1, 2, 3} # Operações com conjuntos frutas.add("uva") # Adiciona elemento frutas.remove("banana") # Remove elemento (gera erro se não existir) frutas.discard("kiwi") # Remove se existir (sem erro) # Operações entre conjuntos a = {1, 2, 3} b = {3, 4, 5} print(a.union(b)) # {1, 2, 3, 4, 5} print(a.intersection(b)) # {3} print(a.difference(b)) # {1, 2}
Dica: Use conjuntos para remover duplicatas de listas: lista_sem_duplicatas = list(set(lista_com_duplicatas))

Fixando o Conhecimento:

Dados os conjuntos A = {1, 2, 3, 4} e B = {3, 4, 5, 6}:

  1. Encontre a união entre A e B
  2. Encontre a interseção entre A e B
  3. Encontre os elementos que estão em A mas não em B
  4. Verifique se A é subconjunto de B
