Módulo 7 - Interfaces

O que são Interfaces?

Interfaces definem contratos que especificam um conjunto de métodos que uma classe deve implementar, garantindo consistência e padronização.

Em Python, não existe uma palavra-chave específica para interfaces, mas podemos usar classes abstratas para definir esses contratos.

Exemplo usando classes abstratas para interface:

from abc import ABC, abstractmethod

class Animal(ABC):
@abstractmethod
def emitir_som(self):
pass

class Cachorro(Animal):
def emitir_som(self):
print("Au au!")

class Gato(Animal):
def emitir_som(self):
print("Miau")

# Testando
def fazer_animal_falar(animal):
animal.emitir_som()

fazer_animal_falar(Cachorro())
fazer_animal_falar(Gato())

Vantagens das Interfaces

  • Forçam a implementação de métodos essenciais.
  • Facilitam a integração entre componentes.
  • Melhoram a organização do código.

Resumo

Embora Python não tenha suporte nativo para interfaces, podemos simular esse conceito com classes abstratas para garantir contratos claros entre classes.
