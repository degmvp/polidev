Documentação Didática: Leitura de Arquivos e Argumentos Variáveis

📌 Introdução

Este exemplo mostra como manipular arquivos de texto com diferentes métodos de leitura, gravação e cópia.
Também demonstra como trabalhar com argumentos variáveis em funções Python.

📂 Abertura e Leitura de Arquivo com open()

print('✅' * 50)
print('''
-------------------------------------------------- 
# ✅ advR003 
# ✅ Python 3.6 alterado: 2018/07/29 
# ✅ Objetivo: Rotina ler arq.txt 
# ✅ Comandos: open(), with open, readlines() 
# ✅ Funções: f.read(), f.readline(), f.readlines() 
--------------------------------------------------''')
print('✅' * 50)

f = open('arq.txt', 'r')

print(f.name)
print(f)
print(f.mode)
lineH = f.readline()
print(lineH)

f.close()

📖 Leitura Completa com with open

print('ler o arquivo a função read() todo o arquivo\n')
with open('arq.txt', 'r') as f:
    f_contents = f.read()
    print(f_contents)

📚 Leitura com readlines()

print('ler o arquivo a função readlines() todo o arquivo\n')
with open('arq.txt', 'r') as f:
    f_contents = f.readlines()
    print(f_contents)

🧾 Leitura com readline() (uma linha)

print('ler o arquivo a função readline() apenas uma linha\n')
with open('arq.txt', 'r') as f:
    f_contents = f.readline()
    print(f_contents, end='')

🔁 Leitura com Loop for

print('ler o arquivo usando for para acesso aos registros\n')
with open('arq.txt', 'r') as f:
    for line in f:
        print(line, end='')

📏 Leitura Parcial com read(size)

print('ler o arquivo usando size explicito 120 bytes\n')
with open('arq.txt', 'r') as f:
    size_to_read = 125

    f_contents = f.read(size_to_read)
    print(f_contents, end='')

    print('imprime o tamanho da variavel size_to_read : ', f.tell())

📤 Copiando Arquivo com write()

print('ler arq.txt e grava saida em arq_copy.txt\n')
with open('arq.txt', 'r') as rf:
    with open('arq_copy.txt', 'w') as wf:
        for line in rf:
            wf.write(line)

🧩 Função com Argumentos Variáveis

A função func_arg usa *args e **kwargs para aceitar qualquer quantidade de argumentos posicionais e nomeados.

def func_arg(*args, **kwargs):
    print(args)
    print(kwargs)

pro = ['Python', 'Programação']
dic = {'nome': 'Degmar', 'desenvolvedor': 'avancado', 'ano ': '2017', 'mes ': 'maio'}

func_arg(*pro, **dic)

✅ Conclusão

A leitura e escrita de arquivos é essencial para automações e manipulação de dados em Python.
O uso de argumentos variáveis torna funções mais flexíveis e reutilizáveis.
