Módulo 11 - List Comprehension em Python

Aprenda a criar listas de forma concisa e eficiente com esta poderosa técnica.

1. O que é List Comprehension

Sintaxe concisa para criar listas baseadas em iteráveis existentes.

Comparação: Tradicional vs Comprehension

# Forma tradicional quadrados = [] for num in range(1, 6): quadrados.append(num ** 2) # Com List Comprehension quadrados = [num ** 2 for num in range(1, 6)]
Vantagens: Mais conciso, legível e geralmente mais rápido.

Fixando o Conhecimento:

Converta estes loops tradicionais para List Comprehension:

# 1. Criar lista com números de 0 a 9 numeros = [] for i in range(10): numeros.append(i) # 2. Converter strings para maiúsculas palavras = ["python", "java", "c++"] maiusculas = [] for palavra in palavras: maiusculas.append(palavra.upper())

2. Exemplo de List Comprehension

Aplicações práticas em diferentes cenários.

Exercício Prático:

# Números pares de 0 a 20 pares = [x for x in range(21) if x % 2 == 0] # Comprimento das palavras palavras = ["python", "comprehension", "list"] tamanhos = [len(p) for p in palavras] # Tuplas com número e seu quadrado quadrados = [(x, x**2) for x in range(5)] # Lista de listas (matriz identidade) identidade = [[1 if i == j else 0 for j in range(3)] for i in range(3)]

Fixando o Conhecimento:

Crie List Comprehensions para:

  1. Gerar os primeiros 10 números cúbicos
  2. Converter temperaturas de Celsius para Fahrenheit
  3. Criar uma matriz 4x4 com valores alternados 0 e 1

4. List Comprehension filtrando com if

Adicionando condições para filtrar elementos.

Exercício Prático:

# Números ímpares entre 0 e 20 impares = [x for x in range(21) if x % 2 != 0] # Palavras com mais de 5 letras frutas = ["maçã", "banana", "kiwi", "laranja"] grandes = [f for f in frutas if len(f) > 5] # Múltiplos de 3 ou 5 multiplos = [x for x in range(50) if x % 3 == 0 or x % 5 == 0] # Condição complexa numeros = [x for x in range(100) if x % 2 == 0 if x % 5 == 0]
Cuidado: List Comprehensions muito complexas podem prejudicar a legibilidade.

Fixando o Conhecimento:

Crie List Comprehensions com filtros para:

  1. Selecionar apenas números positivos de uma lista
  2. Filtrar palavras que começam com vogal
  3. Selecionar anos bissextos entre 1900 e 2100

6. Tratando casos em List Comprehension

Técnicas para lidar com casos especiais e expressões complexas.

Exercício Prático:

# Usando if-else na expressão classificacao = ["Par" if x % 2 == 0 else "Ímpar" for x in range(10)] # Lidando com exceções valores = ["10", "20", "trinta", "40"] numeros = [int(x) if x.isdigit() else 0 for x in valores] # Aninhando comprehensions matriz = [[1, 2], [3, 4], [5, 6]] aplanada = [num for linha in matriz for num in linha] # Dicionário comprehension quadrados = {x: x**2 for x in range(5)}
Dica: Para lógica muito complexa, considere usar funções auxiliares dentro da comprehension.
