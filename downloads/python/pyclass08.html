Módulo 8 - Tuplas em Python

Domine as tuplas, estruturas imutáveis e eficientes do Python.

1. Tuplas

Tuplas são coleções ordenadas e imutáveis de itens.

Exercício Prático:

# Criando tuplas coordenadas = (10, 20) cores = ("vermelho", "verde", "azul") ponto = (3.5, 4.2, "origem") # Acessando elementos print(coordenadas[0]) # 10 print(cores[-1]) # azul # Tuplas são imutáveis # coordenadas[0] = 15 # Erro!
Dica: Use tuplas para dados que não devem ser modificados.

Fixando o Conhecimento:

Crie tuplas para representar:

  1. Coordenadas GPS (latitude, longitude)
  2. Data de nascimento (dia, mês, ano)
  3. RGB de uma cor (red, green, blue)

2. Unpacking em Tuplas

Atribuição desestruturada de valores de tuplas.

Exercício Prático:

# Unpacking básico coordenadas = (10, 20) x, y = coordenadas print(f"X: {x}, Y: {y}") # Unpacking com * valores = (1, 2, 3, 4, 5) primeiro, *meio, ultimo = valores print(primeiro) # 1 print(meio) # [2, 3, 4] print(ultimo) # 5 # Troca de valores a, b = 5, 10 a, b = b, a # Agora a=10, b=5

Fixando o Conhecimento:

Dada a tupla dados = ("João", 25, "Engenheiro", "São Paulo"):

  1. Faça unpacking para variáveis nome, idade, profissão, cidade
  2. Use * para pegar o primeiro e último elemento, empacotando o meio
  3. Crie uma função que retorna tupla e faça unpacking do resultado

3. Lista de Tuplas

Combinação poderosa para estruturar dados.

Exercício Prático:

# Criando lista de tuplas pessoas = [ ("João", 25), ("Maria", 30), ("Carlos", 22) ] # Acessando elementos print(pessoas[0][1]) # Idade do João: 25 # Adicionando nova tupla pessoas.append(("Ana", 28)) # Percorrendo for nome, idade in pessoas: print(f"{nome} tem {idade} anos")

Fixando o Conhecimento:

Crie uma lista de tuplas que represente:

  1. Produtos com (nome, preço, quantidade)
  2. Cidades com (nome, população, estado)
  3. Alunos com (nome, nota1, nota2, nota3)

4. Finalizando Exercício de Lista de Tuplas

Manipulação avançada de listas de tuplas.

Exercício Prático:

alunos = [ ("Ana", 7.5, 8.0, 9.0), ("João", 6.0, 5.5, 7.0), ("Maria", 9.0, 8.5, 10.0) ] # Calculando médias for nome, n1, n2, n3 in alunos: media = (n1 + n2 + n3) / 3 print(f"{nome}: {media:.1f}") # Ordenando por nome alunos.sort() print(alunos) # Ordenando por média (usando função lambda) alunos.sort(key=lambda aluno: (aluno[1]+aluno[2]+aluno[3])/3, reverse=True) print(alunos)

Fixando o Conhecimento:

Dada a lista de produtos produtos = [("Celular", 1200, 5), ("Notebook", 3500, 3), ("Tablet", 900, 8)]:

  1. Calcule o valor total de cada produto (preço × quantidade)
  2. Ordene os produtos por quantidade em estoque
  3. Filtre produtos com valor unitário acima de R$ 1000

5. Exercícios de Tupla Fixação

Exercícios Finais:

  1. Crie um sistema de coordenadas que:
    • Armazene pontos (x, y) em uma lista
    • Calcule a distância entre dois pontos
    • Encontre o ponto mais distante da origem (0,0)
  2. Desenvolva um sistema de notas que:
    • Armazene alunos e 3 notas em tuplas
    • Calcule a média de cada aluno
    • Identifique alunos com média acima de 7
  3. Implemente um tradutor de cores que:
    • Use tuplas para mapear nomes para valores RGB
    • Permita buscar componentes de uma cor
    • Calcule cores intermediárias entre duas cores
