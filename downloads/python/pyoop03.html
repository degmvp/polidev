Módulo 2 - Objetos e Classes

O que são Objetos?

Objetos são instâncias de classes. Eles representam entidades com características (atributos) e comportamentos (métodos).

O que são Classes?

Classes são moldes (ou modelos) que definem como os objetos serão criados, especificando atributos e métodos.

Exemplo prático:

Vamos criar uma classe simples para representar um cachorro.

class Cachorro:
def __init__(self, nome, raca):
self.nome = nome
self.raca = raca

def latir(self):
print(f"{self.nome} está latindo!")

# Criando um objeto (instância) da classe Cachorro
meu_cachorro = Cachorro("Rex", "Labrador")
meu_cachorro.latir()

Atributos e Métodos

  • Atributos: características do objeto (ex: nome, raça).
  • Métodos: ações ou comportamentos do objeto (ex: latir).

Construtor (__init__)

O método __init__ é chamado automaticamente quando um novo objeto é criado. Ele serve para inicializar os atributos do objeto.

Resumo

Neste módulo, aprendemos a diferença entre classes e objetos, criamos uma classe simples com atributos e métodos, e instanciamos objetos a partir dela. No próximo módulo, exploraremos o conceito de encapsulamento para proteger os dados dos objetos.
