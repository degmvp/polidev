Mix-Adv 34 - Tuplas em Python

O que são tuplas?

Tuplas são coleções ordenadas e imutáveis de elementos. Diferente das listas, não podem ser modificadas após a criação.

Como criar tuplas

tupla1 = (1, 2, 3)
tupla2 = ("maçã", "banana", "laranja")
tupla3 = (1, "texto", True, 3.14)
tupla_vazia = ()
tupla_um_elemento = (5,)

Acessando elementos

print(tupla1[0])    # 1
print(tupla2[-1])   # laranja

Tuplas são imutáveis

Você não pode adicionar, remover ou alterar elementos de uma tupla depois de criada.

Uso comum de tuplas

  • Retornar múltiplos valores de uma função.
  • Garantir que dados não sejam alterados acidentalmente.
  • Usar como chaves em dicionários (são hashable).

Desempacotamento

pessoa = ("Ana", 30, "Engenheira")
nome, idade, profissão = pessoa

print(nome)      # Ana
print(idade)     # 30
print(profissão) # Engenheira

Conversão entre listas e tuplas

lista = [1, 2, 3]
tupla = tuple(lista)

tupla2 = (4, 5, 6)
lista2 = list(tupla2)
