Módulo 6 - Abstração

O que é Abstração?

Abstração é o princípio de esconder os detalhes de implementação e expor apenas o necessário. Ajuda a focar no que um objeto faz, e não em como ele faz.

Em Python, podemos aplicar abstração usando classes base e métodos abstratos (com o módulo abc).

Exemplo com classe abstrata:

from abc import ABC, abstractmethod

class Animal(ABC):
@abstractmethod
def emitir_som(self):
pass

class Gato(Animal):
def emitir_som(self):
print("Miau")

class Vaca(Animal):
def emitir_som(self):
print("Muuu")

# Testando
animais = [Gato(), Vaca()]
for a in animais:
a.emitir_som()

Importância da Abstração

  • Reduz a complexidade do sistema.
  • Aumenta a reutilização de código.
  • Facilita a manutenção e evolução.

Resumo

Ao usar abstração, você cria estruturas mais robustas, independentes dos detalhes internos de implementação, promovendo melhores práticas de design de software.
