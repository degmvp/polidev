Módulo 10 - Tratamento de Exceções

O que são Exceções?

Exceções são erros que ocorrem durante a execução do programa. O tratamento adequado das exceções permite que o programa lide com esses erros de forma controlada, evitando travamentos.

Blocos try-except

Permite capturar exceções e tomar ações específicas para tratar erros.

try:
x = 10 / 0
except ZeroDivisionError:
print("Não é possível dividir por zero!")

Capturando exceções genéricas

try:
# código que pode gerar erro
valor = int(input("Digite um número: "))
except Exception as e:
print(f"Ocorreu um erro: {e}")

Bloco else e finally

else: executa se não ocorrer exceção.

finally: executa sempre, independente de exceção.

try:
valor = int(input("Digite um número: "))
except ValueError:
print("Valor inválido!")
else:
print(f"Você digitou: {valor}")
finally:
print("Fim do tratamento.")

Resumo

O tratamento de exceções é fundamental para construir programas robustos, capazes de lidar com erros sem interromper sua execução abruptamente.
