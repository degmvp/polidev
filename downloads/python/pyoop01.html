Polydev OOP - Curso de Programação Orientada a Objetos com Python

Este curso foi desenvolvido para oferecer uma introdução sólida à Programação Orientada a Objetos (POO) utilizando a linguagem Python. Ao longo dos módulos, você encontrará conteúdos explicativos, exemplos práticos e recursos recomendados para apoiar o seu aprendizado.

Recursos do Curso

  • Plataformas de Aprendizagem Online:
    • Coursera – Cursos universitários (ex: University of Michigan)
    • Udemy – Cursos com foco em prática
    • edX – Conteúdos acadêmicos e certificados
    • Alura – Trilhas completas em português
    • Codecademy / SoloLearn – Exercícios interativos
  • Livros Recomendados:
    • "Python Orientado a Objetos" – Rodrigo Turini
    • "Python 3 Object-Oriented Programming" – Dusty Phillips
    • "Clean Code" – Robert C. Martin
    • "Design Patterns em Python"
  • Ambientes de Desenvolvimento Integrado (IDEs):
    • PyCharm – IDE robusta com recursos completos
    • VS Code – Leve e com excelente suporte a Python
    • Jupyter Notebooks – Ideal para ensino e visualização
