Módulo 4 - Herança

O que é Herança?

Herança é o mecanismo que permite criar novas classes baseadas em classes já existentes. Isso promove a reutilização de código e facilita a extensão de funcionalidades.

Vantagens da Herança

  • Reutilização de código.
  • Facilidade de manutenção e expansão do sistema.
  • Organização hierárquica entre classes.

Exemplo prático:

Vamos criar uma classe base Animal e duas subclasses: Cachorro e Gato.

class Animal:
def __init__(self, nome):
self.nome = nome
def falar(self):
return "Som genérico"

class Cachorro(Animal):
def falar(self):
return "Au au!"

class Gato(Animal):
def falar(self):
return "Miau"

# Testando
a = Animal("Bicho")
print(a.falar()) # Som genérico
c = Cachorro("Rex")
print(c.falar()) # Au au!
g = Gato("Mimi")
print(g.falar()) # Miau

Uso do super()

É possível acessar métodos e atributos da classe pai usando super().

class Pessoa:
def __init__(self, nome):
self.nome = nome

class Estudante(Pessoa):
def __init__(self, nome, curso):
super().__init__(nome)
self.curso = curso

e = Estudante("Ana", "Python")
print(e.nome) # Ana
print(e.curso) # Python

Resumo

Herança permite criar estruturas hierárquicas e reaproveitar funcionalidades entre classes. No próximo módulo, abordaremos o polimorfismo, que amplia ainda mais o poder da herança.
