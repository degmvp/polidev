Mix-Adv 26 - Estruturas Condicionais (if, elif, else)

Introdução

As estruturas condicionais são utilizadas para executar blocos de código de acordo com uma condição booleana.

Sintaxe do if

if condição:
    # bloco se verdadeiro

Exemplo:

idade = 18

if idade >= 18:
    print("Maior de idade.")

if ... else

nota = 6

if nota >= 7:
    print("Aprovado")
else:
    print("Reprovado")

if ... elif ... else

media = 5

if media >= 7:
    print("Aprovado")
elif media >= 5:
    print("Recuperação")
else:
    print("Reprovado")

Importante

O uso correto da indentação (espaços à esquerda) é obrigatório em Python e indica blocos de código.
