Mix-Adv 31 - Escopo de Variáveis em Funções

O que é escopo?

Escopo se refere ao contexto onde uma variável é reconhecida e acessível. Existem dois tipos principais de escopo em Python: local e global.

Escopo local

def minha_funcao():
    x = 10  # x é local
    print(x)

minha_funcao()
# print(x)  # Isso causaria erro, pois x não existe fora da função

Escopo global

x = 5  # variável global

def mostrar():
    print(x)

mostrar()  # Saída: 5

Modificando variáveis globais

x = 10

def alterar():
    global x
    x = 20

alterar()
print(x)  # Saída: 20

A palavra-chave global permite modificar uma variável definida fora da função.

Variáveis com mesmo nome

x = 50

def teste():
    x = 100  # nova variável local x
    print("Dentro da função:", x)

teste()
print("Fora da função:", x)

A variável local x dentro da função não altera a x global.

Resumo

  • Variáveis locais existem apenas dentro da função onde são definidas.
  • Variáveis globais são definidas fora de funções e acessíveis em todo o programa.
  • Use global com cuidado, pois altera variáveis globais dentro de funções.
