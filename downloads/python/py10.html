Documentação Didática: Operações Bitwise e Controle de Permissões

📌 Introdução

Este exemplo mostra como usar operações bit a bit (bitwise) em Python. Também aplicamos essas operações para controlar permissões de usuários em um sistema.

🔁 Conversões Binárias e Hexadecimais

print("9 em binário -->", bin(9))
print("0b1001 em decimal -->", int("0b1001", 2))

print("147 em hexadecimal -->", hex(147))
print("0x93 em decimal -->", int("0x93", 16))

➡️ Operadores de Shift

print("0b0010100 >> 2 -->", 0b0010100 >> 2)
print("0b0010100 << 2 -->", 0b0010100 << 2)

⚙️ Operações Bitwise

As operações bitwise comparam bits de dois valores. São comuns em programação de baixo nível e manipulação de permissões.

AND (&)

a = 42  # 00101010
b = 15  # 00001111
print("a & b -->", a & b)  # 00001010 = 10

OR (|)

print("a | b -->", a | b)  # 00101111 = 47

XOR (^)

print("a ^ b -->", a ^ b)  # 00100101 = 37

🔐 Controle de Permissões com Bits

Utilizando bits para representar permissões:

  • view = 1
  • edit = 2
  • create = 4

class User:
    def __init__(self, rights):
        self.rights = rights

    def can_view(self):
        return view & self.rights

    def can_edit(self):
        return edit & self.rights

    def can_create(self):
        return create & self.rights

👥 Usuários e Acessos

usr = User(view)
adm = User(view | edit)
dba = User(view | edit | create)

print(usr.can_view())    # True
print(usr.can_edit())    # False
print(adm.can_edit())    # True
print(dba.can_create())  # True

✅ Conclusão

A manipulação de bits é útil em aplicações que exigem controle detalhado de permissões ou representação compacta de estados.
