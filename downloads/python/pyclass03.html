Módulo 3 - Condicionais em Python (If)

Controle o fluxo do seu programa com estruturas condicionais.

1. Estrutura do if - Condições no Python

O if executa um bloco de código apenas se uma condição for verdadeira.

Exercício Prático:

idade = 18 if idade >= 18: print("Você é maior de idade!") print("Pode acessar o conteúdo.")
Observação: Todo o código indentado pertence ao bloco do if!

Fixando o Conhecimento:

Crie uma condição que:

  1. Verifique se um número é positivo
  2. Se for, imprima "Número positivo"

2. Blocos e Indentação - If dentro de If

Podemos aninhar condições colocando um if dentro de outro.

Exercício Prático:

numero = 15 if numero > 10: print("Número maior que 10") if numero > 20: print("E também maior que 20!") print("Ainda dentro do primeiro if")
Cuidado: A indentação define quais instruções pertencem a qual bloco!

Fixando o Conhecimento:

Crie um programa que:

  1. Pergunte a idade do usuário
  2. Se for maior que 12, verifique se é maior que 18
  3. Mostre mensagens diferentes para cada faixa etária

3. Elif

elif testa condições adicionais quando o if anterior é falso.

Exercício Prático:

nota = 7.5 if nota >= 9: print("Conceito A") elif nota >= 7: print("Conceito B") elif nota >= 5: print("Conceito C") else: print("Conceito D")
Fluxo: Python testa cada condição em ordem até encontrar uma verdadeira.

Fixando o Conhecimento:

Crie um programa que:

  1. Pergunte a temperatura atual
  2. Classifique como:
    • "Muito quente" se > 30°C
    • "Quente" se > 20°C
    • "Ameno" se > 10°C
    • "Frio" se ≤ 10°C

4. Comparadores

Operadores para comparar valores: ==, !=, >, <, >=, <=

Exercício Prático:

a = 5 b = 7 print(a == b) # Igual print(a != b) # Diferente print(a > b) # Maior que print(a < b) # Menor que print(a >= 5) # Maior ou igual print(b <= 10) # Menor ou igual

Fixando o Conhecimento:

Complete as condições:

idade = 15 # Verifique se a idade é diferente de 18 if idade __ 18: print("Idade não é 18") # Verifique se a idade é entre 13 e 19 (inclusive) if idade __ 13 __ idade __ 19: print("Adolescente")

5. And e Or

Combinar condições com operadores lógicos.

Exercício Prático:

tem_sol = True tem_guarda_sol = False # AND - ambas condições devem ser verdadeiras if tem_sol and tem_guarda_sol: print("Pode ir à praia com proteção") # OR - pelo menos uma condição verdadeira if tem_sol or tem_guarda_sol: print("Talvez possa ir à praia")
Dica: and tem precedência sobre or (use parênteses para controlar).

Fixando o Conhecimento:

Crie condições que:

  1. Verifiquem se um número está entre 10 e 20 OU maior que 50
  2. Verifiquem se uma pessoa pode votar (idade ≥ 16 E tem título)

6. Exercícios If - Fixação

Exercícios Finais:

  1. Crie um programa que:

    • Mostre qual é o maior ou se são iguais

    • Verifique se correspondem aos valores esperados

    • Pergunte dois números e uma operação (+, -, *, /)
