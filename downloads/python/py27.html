Mix-Adv 27 - Laços de Repetição: while

Introdução

O laço while permite repetir um bloco de código enquanto uma condição for verdadeira.

Sintaxe

while condição:
    # bloco de código a ser repetido

Exemplo Básico

contador = 1

while contador <= 5:
    print("Contador:", contador)
    contador += 1

Loop Infinito

Se a condição nunca for falsa, o laço entra em loop infinito: 

while True:
    comando = input("Digite 'sair' para encerrar: ")
    if comando == "sair":
        break

Comando break

break é usado para interromper o laço mesmo que a condição ainda seja verdadeira.

Comando continue

continue pula a iteração atual e vai para a próxima repetição do laço. 

num = 0

while num < 10:
    num += 1
    if num == 5:
        continue
    print(num)
