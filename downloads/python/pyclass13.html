Módulo 13 - Funções como Parâmetros e Expressões Lambda

Domine técnicas avançadas de programação funcional em Python.

1. Functions em Iterables e a função map

Aplicando funções a cada elemento de uma sequência com map().

Comparação: Tradicional vs map

# Forma tradicional numeros = [1, 2, 3, 4] quadrados = [] for num in numeros: quadrados.append(num ** 2) # Com map def quadrado(x): return x ** 2 quadrados = list(map(quadrado, numeros)) # Com lambda quadrados = list(map(lambda x: x**2, numeros))
Dica: map retorna um iterador - converta para lista quando necessário.

Fixando o Conhecimento:

Use map para:

  1. Converter lista de strings para maiúsculas
  2. Calcular raiz quadrada de números
  3. Aplicar uma função de formatação em valores monetários

2. Aplicando function em iterable no sort

Personalizando a ordenação com o parâmetro key.

Exercício Prático:

# Ordenação personalizada pessoas = [ {"nome": "Ana", "idade": 25}, {"nome": "Carlos", "idade": 30}, {"nome": "Beatriz", "idade": 22} ] # Ordenando por nome pessoas.sort(key=lambda p: p["nome"]) print(pessoas) # Ordenando por idade (decrescente) pessoas.sort(key=lambda p: p["idade"], reverse=True) print(pessoas) # Ordenação complexa produtos = [("Mouse", 50), ("Teclado", 120), ("Monitor", 800)] produtos.sort(key=lambda x: x[1]) # Ordena por preço

Fixando o Conhecimento:

Ordene estas listas:

  1. Lista de palavras pelo comprimento
  2. Lista de tuplas (nome, nota) pela nota
  3. Lista de dicionários por múltiplas chaves

3. Lambda Expressions

Funções anônimas para operações simples.

Exercício Prático:

# Sintaxe básica # lambda argumentos: expressão dobro = lambda x: x * 2 print(dobro(5)) # 10 # Múltiplos argumentos soma = lambda a, b: a + b print(soma(3, 7)) # 10 # Uso imediato print((lambda x: x.upper())("python")) # PYTHON
Cuidado: Lambdas devem ser simples - para lógica complexa, use funções nomeadas.

Fixando o Conhecimento:

Crie lambdas para:

  1. Verificar se número é par
  2. Concatenar duas strings com espaço entre elas
  3. Calcular IMC (peso / altura²)

4. A grande utilidade de Lambda Expressions

Casos de uso comuns onde lambdas brilham.

Exercício Prático:

# Com filter() numeros = [1, 2, 3, 4, 5, 6] pares = list(filter(lambda x: x % 2 == 0, numeros)) # Com sorted() palavras = ["python", "java", "c", "javascript"] ordenadas = sorted(palavras, key=lambda x: len(x)) # Em interfaces gráficas (exemplo simplificado) # botao.config(command=lambda: funcao(parametro)) # Em reduções (functools.reduce) from functools import reduce soma = reduce(lambda x, y: x + y, numeros)

Fixando o Conhecimento:

Use lambdas com:

  1. filter para selecionar palavras com mais de 5 letras
  2. sorted para ordenar dicionários por valor
  3. reduce para calcular o produto de números

5. Lambda Expression para criar um Construtor de Funções

Fábricas de funções dinâmicas com lambdas.

Exercício Prático:

# Construtor de funções de potência def criar_potencia(expoente): return lambda base: base ** expoente quadrado = criar_potencia(2) cubo = criar_potencia(3) print(quadrado(5)) # 25 print(cubo(3)) # 27 # Construtor de filtros def criar_filtro(limite): return lambda x: x > limite filtrar_maiores_10 = criar_filtro(10) print(filtrar_maiores_10(15)) # True

Fixando o Conhecimento:

Crie construtores que:

  1. Gerem funções de multiplicação por n
  2. Produzam verificadores de divisibilidade
  3. Criem formatadores de texto com prefixo configurável
