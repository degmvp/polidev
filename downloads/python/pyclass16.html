1. Python e Integrações de API

Python oferece várias bibliotecas para trabalhar com APIs RESTful e JSON:

import requests
import json

# Exemplo básico de requisição API
response = requests.get('https://api.exemplo.com/dados')
data = response.json()

print(json.dumps(data, indent=2))
Bibliotecas úteis:
  • requests - Para fazer requisições HTTP
  • json - Para trabalhar com dados JSON
  • http.client - Cliente HTTP de baixo nível
  • urllib - Alternativa ao requests

2. API de Cotação de Moeda

Exemplo usando a API do Banco Central ou AwesomeAPI:

API: AwesomeAPI (Cotação atual)

Endpoint: https://economia.awesomeapi.com.br/json/last/USD-BRL

import requests

def get_currency_rate(from_currency, to_currency):
    url = f'https://economia.awesomeapi.com.br/json/last/{from_currency}-{to_currency}'
    response = requests.get(url)
    data = response.json()
    return float(data[f'{from_currency}{to_currency}']['bid'])

# Uso
usd_to_brl = get_currency_rate('USD', 'BRL')
print(f'1 USD = {usd_to_brl:.2f} BRL')

3. API de CEP e Busca de Endereços

Exemplo usando a API ViaCEP:

API: ViaCEP

Endpoint: https://viacep.com.br/ws/01001000/json/

def get_address_by_cep(cep):
    url = f'https://viacep.com.br/ws/{cep}/json/'
    response = requests.get(url)
    
    if response.status_code == 200:
        return response.json()
    else:
        return None

# Uso
address = get_address_by_cep('01001000')
if address:
    print(f"Endereço: {address['logradouro']}, {address['localidade']}/{address['uf']}")
else:
    print("CEP não encontrado")

4. API de Preço de Ações e Informações Econômicas

Exemplo usando a Alpha Vantage (requer chave API):

API: Alpha Vantage

Endpoint: https://www.alphavantage.co/query?function=TIME_SERIES_DAILY&symbol=IBM&apikey=SEU_API_KEY

def get_stock_data(symbol, api_key):
    url = f'https://www.alphavantage.co/query?function=TIME_SERIES_DAILY&symbol={symbol}&apikey={api_key}'
    response = requests.get(url)
    return response.json()

# Uso (substitua 'SUA_CHAVE' por uma chave API válida)
stock_data = get_stock_data('IBM', 'SUA_CHAVE')
latest_date = list(stock_data['Time Series (Daily)'].keys())[0]
latest_price = stock_data['Time Series (Daily)'][latest_date]['4. close']
print(f"Preço mais recente de IBM: ${latest_price}")

Melhores Práticas para Trabalhar com APIs

  • Sempre verifique o status code da resposta
  • Trate erros e exceções (timeout, limites de requisição)
  • Use headers apropriados (Content-Type, Authorization)
  • Respeite os limites de requisição (rate limiting)
  • Armazene chaves API em variáveis de ambiente
  • Considere usar cache para respostas

Exemplo: Tratamento de Erros

try:
    response = requests.get(url, timeout=5)
    response.raise_for_status()  # Levanta exceção para códigos 4xx/5xx
    data = response.json()
except requests.exceptions.RequestException as e:
    print(f"Erro na requisição: {e}")
except ValueError as e:
    print(f"Erro ao decodificar JSON: {e}")
