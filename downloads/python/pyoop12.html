Módulo 11 - Coleções

O que são Coleções?

Coleções são estruturas de dados que armazenam e organizam múltiplos objetos, facilitando sua manipulação e acesso.

Principais tipos de coleções em Python

  • Lista: coleção ordenada e mutável.
  • Tupla: coleção ordenada e imutável.
  • Conjunto (set): coleção não ordenada, sem elementos duplicados.
  • Dicionário: coleção de pares chave-valor.

Exemplos práticos

# Lista
frutas = ["maçã", "banana", "laranja"]
frutas.append("uva")
print(frutas) # ['maçã', 'banana', 'laranja', 'uva']

# Tupla
coordenadas = (10, 20)
print(coordenadas)

# Conjunto
numeros = {1, 2, 3, 2, 1}
print(numeros) # {1, 2, 3}

# Dicionário
pessoa = {"nome": "Ana", "idade": 30}
print(pessoa["nome"]) # Ana

Coleções e POO

Em POO, coleções são usadas para armazenar múltiplos objetos, como listas de instâncias de uma classe.

class Aluno:
def __init__(self, nome):
self.nome = nome

alunos = [Aluno("João"), Aluno("Maria"), Aluno("Pedro")]
for aluno in alunos:
print(aluno.nome)

Resumo

Conhecer e utilizar coleções é fundamental para manipular grupos de objetos em POO e em programação geral.
