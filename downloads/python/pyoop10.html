Módulo 9 - Relacionamentos entre Objetos

Tipos de Relacionamentos

Na POO, os objetos podem se relacionar entre si de várias formas, principalmente através de:

  • Agregação: relacionamento onde um objeto contém outro, mas o objeto contido pode existir independentemente.
  • Composição: relacionamento forte onde o objeto contido não pode existir sem o objeto que o contém.

Exemplo de Agregação

Um objeto Departamento possui vários Funcionarios, mas os funcionários podem existir sem o departamento.

class Funcionario:
def __init__(self, nome):
self.nome = nome

class Departamento:
def __init__(self, nome):
self.nome = nome
self.funcionarios = []

def adicionar_funcionario(self, funcionario):
self.funcionarios.append(funcionario)

# Teste
f1 = Funcionario("Ana")
f2 = Funcionario("Carlos")
dep = Departamento("RH")
dep.adicionar_funcionario(f1)
dep.adicionar_funcionario(f2)

Exemplo de Composição

Um objeto Carro é composto por objetos Motor. O motor não existe sem o carro.

class Motor:
def __init__(self, potencia):
self.potencia = potencia

class Carro:
def __init__(self, modelo, potencia_motor):
self.modelo = modelo
self.motor = Motor(potencia_motor)

# Teste
carro = Carro("Fusca", 120)
print(carro.motor.potencia) # Saída: 120

Resumo

Agregação e composição definem como objetos se relacionam, com diferentes graus de dependência e ciclo de vida. Compreender esses relacionamentos ajuda a modelar sistemas mais realistas e eficientes.
