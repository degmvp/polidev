Módulo 1 - Início em Python

Neste módulo, vamos aprender Python de forma prática, colocando a mão no código desde o primeiro momento.

1. Criando seu Primeiro Programa

Em Python, usamos a função print() para exibir informações na tela.

Exercício Prático:

Execute o código abaixo no seu ambiente Python:

print("Minha primeira linha de código Python!")
Dica: No Jupyter Notebook, basta digitar o código em uma célula e pressionar Shift+Enter para executar.

Fixando o Conhecimento:

Modifique o código para exibir uma mensagem diferente:

# Seu código aqui print("Python é poderoso!")

2. Operações Básicas

Python pode ser usado como uma calculadora poderosa. Vamos testar as operações básicas:

Exercício Prático:

print(5 + 3) # Adição print(10 - 2) # Subtração print(4 * 2) # Multiplicação print(16 / 2) # Divisão

Fixando o Conhecimento:

Complete os códigos abaixo com os operadores corretos (+, -, *, /):

print(10 __ 2) # Deve resultar em 5 print(7 __ 3) # Deve resultar em 21 print(15 __ 3) # Deve resultar em 12 print(20 __ 4) # Deve resultar em 5

3. Ordem das Operações no Python

Python segue a ordem matemática padrão de operações (PEMDAS): Parênteses, Expoentes, Multiplicação/Divisão, Adição/Subtração.

Exercício Prático:

print(2 + 3 * 4) # Resultado: 14 (multiplicação primeiro) print((2 + 3) * 4) # Resultado: 20 (parênteses primeiro)

Fixando o Conhecimento:

Preveja o resultado antes de executar:

print(10 + 20 / 5 - 3) print(100 / (10 * 2)) print(2 ** 3 + 1) # O que é 2**3?
Desafio: Escreva uma expressão que some 5 e 7, depois multiplique o resultado por 3.

4. Operações com Texto (String)

Em Python, textos são chamados de strings. Podemos concatenar (juntar) e repetir strings.

Exercício Prático:

print("Python" + " " + "é" + " legal!") # Concatenação print("Repetir " * 3) # Repetição

Fixando o Conhecimento:

Crie um programa que:

  1. Concatene seu nome e sobrenome
  2. Repita a palavra "Olá" 3 vezes
  3. Concatene "Python" com o número 3 (dica: use str(3))

5. Variáveis

Variáveis são como caixas que guardam informações para usarmos depois.

Exercício Prático:

linguagem = "Python" versao = 3.9 print(linguagem) print("Versão:", versao)

Fixando o Conhecimento:

Crie variáveis para:

  1. Seu nome completo
  2. Sua idade
  3. Seu ano de nascimento (calcule com base na idade)
  4. Exiba todas as informações juntas

6. Input - Pegando informações do Usuário

Podemos interagir com o usuário usando a função input().

Exercício Prático:

nome = input("Qual seu nome? ") print("Olá,", nome, "! Bem-vindo ao Python!")

Fixando o Conhecimento:

Crie um programa que:

  1. Pergunte a idade do usuário
  2. Calcule quantos dias aproximadamente ele já viveu (idade * 365)
  3. Exiba o resultado
Lembrete: O input sempre retorna uma string. Use int() para converter para número.

7. Os arquivos do Jupyter e como abrir nossos arquivos

Para salvar seu trabalho no Jupyter Notebook:

  1. Clique em "File" > "Save and Checkpoint" para salvar

  3. Para abrir novamente, inicie o Jupyter e navegue até o arquivo
