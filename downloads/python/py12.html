✅ Introdução

Este exemplo implementa um gerenciador de telas com múltiplas páginas (multi-frame) usando Tkinter em Python. O sistema é baseado em uma classe principal que empilha páginas como StartPage, PageOne, PageTwo e Page3.

✅ Objetivo

Criar um container com várias telas, que podem ser alternadas por botões. Cada tela tem funcionalidades próprias como ler arquivos de texto e exibir em canvas com scrollbar.

✅ Estrutura Principal


class SampleApp(tk.Tk):
    def __init__(self, *args, **kwargs):
        ...
        self.frames = {}
        for F in (StartPage, PageOne, PageTwo, Page3):

        self.show_frame("StartPage")

    def show_frame(self, page_name):
        frame = self.frames[page_name]
        frame.tkraise()

✅ Tela Inicial (StartPage)

A tela inicial exibe botões para navegar entre as outras telas e encerrar o programa. 


class StartPage(tk.Frame):
    def __init__(self, parent, controller):
        ...
        button1 = tk.Button(..., command=lambda: controller.show_frame("PageOne"))
        button2 = tk.Button(..., command=lambda: controller.show_frame("PageTwo"))
        button3 = tk.Button(..., command=exit)
