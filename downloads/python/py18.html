✅ Introdução

Este exemplo demonstra como implementar um decorator com memorization (armazenamento em cache) para otimizar chamadas repetidas de funções. A classe `memorize` atua como um dicionário que armazena os resultados das chamadas para evitar recomputações.

✅ Mensagem de Cabeçalho


print('✅' * 50)
print(''' 
#--------------------------------------- 
# ✅ AppB03 
# ✅ Python 3.6 alterado: 2018/08/14 
# ✅ Objetivo: usando decorator --> @memorize 
#---------------------------------------
''')
print('✅' * 50)

✅ Classe Decorator memorize


class memorize(dict):
    def __init__(self, func):
        self.func = func

    def __call__(self, *args):
        return self[args]

    def __missing__(self, key):
        result = self[key] = self.func(*key)
        return result

✅ Função decorada com @memorize


@memorize
def foo(a, b):
    return a * b

✅ Chamadas e Resultados

Ao chamar a função `foo` várias vezes com os mesmos parâmetros, o resultado será obtido do cache interno do decorator sem recalcular. 


print('Efetua 3 operações com a função foo(a, b): 2x4, 12x4, 12x12')
print('Armazena os resultados no decorador memorize (dicionário)')
print(foo(2, 4))     # 8
print(foo(12, 4))    # 48
print(foo(12, 12))   # 144
print(foo)           # mostra o objeto memorize
