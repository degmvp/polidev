Mix-Adv 29 - Funções em Python

Introdução

Funções são blocos de código reutilizáveis que realizam uma tarefa específica. Em Python, funções são definidas com a palavra-chave def.

Sintaxe de uma função

def nome_da_funcao(parâmetros):
    # bloco de código
    return resultado

Exemplo de função simples

def saudacao():
    print("Olá, bem-vindo ao curso de Python!")

saudacao()

Função com parâmetros

def somar(a, b):
    return a + b

resultado = somar(5, 3)
print("Soma:", resultado)

Parâmetro com valor padrão

def apresentar(nome, saudacao="Olá"):
    print(saudacao, nome)

apresentar("Carlos")
apresentar("Ana", "Oi")

Função com retorno

A instrução return é usada para devolver um valor ao ponto onde a função foi chamada.

Funções como objetos

Em Python, funções podem ser passadas como argumentos, atribuídas a variáveis e armazenadas em estruturas de dados. 

def dobro(x):
    return x * 2

operacao = dobro
print(operacao(10))
