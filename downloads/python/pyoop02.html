Módulo 1 - Introdução à Programação Orientada a Objetos

O que é POO?

A Programação Orientada a Objetos (POO) é um paradigma de programação que utiliza objetos como base da construção dos programas.

Esses objetos combinam dados (atributos) e comportamentos (métodos), promovendo organização, reutilização de código e modularidade.

Histórico

O paradigma surgiu na década de 1960 com a linguagem Simula e ganhou força com Smalltalk, C++, Java e Python. Atualmente, é amplamente utilizado no desenvolvimento de sistemas complexos.

Comparação com a Programação Estruturada

Na programação estruturada, o foco está em funções e no fluxo do programa. Na POO, o foco está em entidades chamadas objetos, que representam elementos do mundo real.

Exemplo comparativo

Programação Estruturada:

def dirigir(carro):
print(f"{carro} está em movimento")

dirigir("Fusca")

Programação Orientada a Objetos:

class Carro:
def __init__(self, modelo):
self.modelo = modelo

def dirigir(self):
print(f"{self.modelo} está em movimento")

fusca = Carro("Fusca")
fusca.dirigir()

Benefícios da POO

  • Reutilização de código: através de herança e composição.
  • Organização: código mais limpo, coeso e fácil de entender.
  • Facilidade de manutenção: módulos independentes facilitam ajustes.
  • Segurança: encapsulamento protege os dados internos.

Resumo

Este módulo introduziu os conceitos básicos da POO, comparando com a programação tradicional e mostrando um exemplo prático. Nos próximos módulos, veremos como criar nossas próprias classes, encapsular dados, herdar comportamentos e muito mais.
