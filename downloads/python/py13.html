✅ Introdução

Este exemplo demonstra como usar o módulo tkinter para criar uma interface gráfica que se conecta a um banco de dados MySQL, permitindo inserções e visualização de dados.

✅ Requisitos

É necessário ter o MySQL Server instalado e o módulo mysql.connector disponível no Python: 

pip install mysql-connector-python

✅ Conexão com MySQL


import mysql.connector

mydb = mysql.connector.connect(
    host="localhost",
    user="root",
    password="rsx45831",
    database="polydev"
)
mycursor = mydb.cursor()

✅ Interface Gráfica com Tkinter

A interface principal permite digitar um código e nome, que são então inseridos na tabela linguagens do banco de dados. 


from tkinter import *

root = Tk()
root.title("App Entrada Banco")
root.geometry("400x180")

✅ Função de Inserção


def inserir():
    cod = e1.get()
    nome = e2.get()
    mycursor.execute("INSERT INTO linguagens (id, nome) VALUES (%s, %s)", (cod, nome))
    mydb.commit()
    e1.delete(0, END)
    e2.delete(0, END)

✅ Widgets e Layout


Label(root, text="Código:").pack()
e1 = Entry(root)
e1.pack()

Label(root, text="Nome:").pack()
e2 = Entry(root)
e2.pack()

Button(root, text="Inserir", command=inserir).pack(pady=10)

✅ Execução


root.mainloop()

Com esta aplicação, você pode inserir registros diretamente na sua base de dados polydev sem necessidade de usar comandos SQL manuais.
