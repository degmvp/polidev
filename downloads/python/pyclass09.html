Módulo 9 - Dicionários em Python

Aprenda a trabalhar com estruturas chave-valor, uma das mais poderosas do Python.

1. Dicionários em Python

Dicionários armazenam pares chave-valor em uma estrutura mutável e não ordenada.

Exercício Prático:

# Criando dicionários pessoa = { "nome": "João Silva", "idade": 30, "cidade": "São Paulo" } produto = { "nome": "Notebook", "preco": 3500.00, "estoque": True } # Acessando valores print(pessoa["nome"]) # João Silva print(produto.get("preco")) # 3500.0
Dica: Use dicionários quando precisar de acesso rápido a valores através de chaves descritivas.

Fixando o Conhecimento:

Crie dicionários para representar:

  1. Seus dados pessoais (nome, idade, cidade)
  2. Um produto com pelo menos 5 características
  3. Um filme com informações como título, ano e diretor

2. Pegar item Dicionário e Verificar Item Dicionário

Formas seguras de acessar e verificar a existência de itens.

Exercício Prático:

aluno = { "nome": "Maria", "nota": 8.5, "aprovado": True } # Acesso direto (pode gerar KeyError) # print(aluno["idade"]) # Acesso seguro com get() print(aluno.get("idade", "Não informado")) # Verificando existência de chave if "nota" in aluno: print(f"Nota: {aluno['nota']}") # Método setdefault() aluno.setdefault("faltas", 0) print(aluno)

Fixando o Conhecimento:

Dado o dicionário livro = {"titulo": "Python Essencial", "autor": "Carlos Coder", "ano": 2020}:

  1. Verifique se existe a chave "editora"
  2. Acesse o autor de forma segura (com get)
  3. Adicione "páginas" com valor padrão 300 se não existir

3. Adicionar, Modificar e Remover Itens do Dicionário

Manipulando o conteúdo de dicionários.

Exercício Prático:

config = { "tema": "escuro", "notificacoes": True } # Adicionando itens config["idioma"] = "pt-br" # Modificando valores config["tema"] = "claro" # Removendo itens valor_removido = config.pop("notificacoes") del config["idioma"] # Limpando o dicionário config.clear()
Cuidado: del e pop() geram KeyError se a chave não existir.

Fixando o Conhecimento:

Crie um programa que:

  1. Comece com dicionário vazio
  2. Permita adicionar pares chave-valor
  3. Permita remover itens por chave
  4. Mostre o dicionário atualizado após cada operação

4. For em Dicionário

Iterando sobre chaves, valores ou itens de um dicionário.

Exercício Prático:

estoque = { "mouse": 50, "teclado": 30, "monitor": 15, "headset": 20 } # Iterando sobre chaves for produto in estoque: print(produto) # Iterando sobre valores for quantidade in estoque.values(): print(quantidade) # Iterando sobre itens (chave e valor) for produto, qtd in estoque.items(): print(f"{produto}: {qtd} unidades")

Fixando o Conhecimento:

Dado o dicionário notas = {"Matemática": 8.5, "Português": 7.0, "História": 9.0}:

  1. Percorra e mostre todas as disciplinas
  2. Calcule a média das notas
  3. Encontre a disciplina com maior nota

6. Métodos Úteis Dicionários (Parte 1) - items

O método items() retorna uma visão dos pares chave-valor.

Exercício Prático:

aluno = { "nome": "Ana", "curso": "Engenharia", "semestre": 3 } # Convertendo items em lista itens = list(aluno.items()) print(itens) # [('nome', 'Ana'), ('curso', 'Engenharia'), ('semestre', 3)] # Atualizando múltiplos valores aluno.update({"semestre": 4, "bolsista": True}) print(aluno) # Criando cópia copia_aluno = aluno.copy()

Fixando o Conhecimento:

Dado o dicionário config = {"tema": "claro", "fonte": "Arial", "tamanho": 12}:

  1. Use items() para percorrer e mostrar todas configurações
  2. Atualize múltiplas configurações de uma vez
  3. Crie uma cópia profunda do dicionário

7. Métodos Úteis Dicionários (Parte 2) - keys e values

Métodos para acessar chaves e valores separadamente.

Exercício Prático:

funcionario = { "matricula": "F123", "cargo": "Analista", "salario": 4500.00 } # Obtendo listas de chaves e valores chaves = list(funcionario.keys()) valores = list(funcionario.values()) print(chaves) # ['matricula', 'cargo', 'salario'] print(valores) # ['F123', 'Analista', 4500.0] # Verificando valores if 4500.00 in funcionario.values(): print("Salário encontrado")

Fixando o Conhecimento:

Crie um programa que:

  1. Percorra apenas as chaves de um dicionário e as ordene alfabeticamente
  2. Verifique se um determinado valor existe no dicionário
  3. Conte quantas vezes um valor aparece no dicionário

8. Zip e Transformando Listas em Dicionários

Convertendo entre listas e dicionários usando zip().

Exercício Prático:

# Criando dicionário a partir de listas chaves = ["nome", "idade", "cidade"] valores = ["Carlos", 25, "Belo Horizonte"] pessoa = dict(zip(chaves, valores)) print(pessoa) # {'nome': 'Carlos', 'idade': 25, 'cidade': 'Belo Horizonte'} # Zip com múltiplas listas produtos = ["mouse", "teclado", "monitor"] precos = [50.00, 120.00, 800.00] estoque = [30, 25, 10] inventario = {prod: (preco, qtd) for prod, preco, qtd in zip(produtos, precos, estoque)} print(inventario)

Fixando o Conhecimento:

Dadas as listas:

disciplinas = ["Matemática", "Português", "História"] notas = [8.5, 7.0, 9.0] professores = ["Carlos", "Ana", "Márcio"]

Crie:

  1. Um dicionário {disciplina: nota}
  2. Um dicionário {disciplina: (nota, professor)}
  3. Uma lista de dicionários com todas informações
