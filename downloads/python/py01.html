Documentação Didática: Fundamentos de Classe em Python

📌 Introdução

Este código apresenta a estrutura básica de uma classe Python com atributos e métodos simples.

🔧 Definição da Classe Pessoa

A classe Pessoa possui dois atributos: nome e idade. Um método chamado mostrar_dados exibe essas informações.

class Pessoa:
    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade

    def mostrar_dados(self):
        print(f"Nome: {self.nome}")
        print(f"Idade: {self.idade}")

p1 = Pessoa("João", 30)
p1.mostrar_dados()

✅ Explicação Final

O método __init__ é o construtor da classe, chamado automaticamente na criação de um novo objeto.
O uso de self permite acessar os atributos e métodos do próprio objeto.
