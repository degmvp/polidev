Módulo 4 - Strings em Python

Domine a manipulação de textos, uma das habilidades mais importantes em Python.

1. Por que aprender Strings e a importância para Python

Strings são essenciais em praticamente todos os programas Python.

Exercício Prático:

# Strings estão em todo lugar! nome = "João Silva" email = "joao@exemplo.com" endereco = "Rua Python, 123" telefone = "(11) 98765-4321" print("Cliente:", nome) print("E-mail:", email)
Dica: Em Python, tudo entre aspas simples (' ') ou duplas (" ") é uma string.

Fixando o Conhecimento:

Crie variáveis string para armazenar:

  1. Seu nome completo
  2. Seu endereço
  3. Um CPF (como texto, não número)
  4. Exiba todas as informações formatadas

2. Índice e Tamanho de String

Podemos acessar caracteres individuais e medir o comprimento da string.

Exercício Prático:

linguagem = "Python" # Acessando caracteres por índice (começa em 0) print(linguagem[0]) # 'P' print(linguagem[2]) # 't' # Tamanho da string print(len(linguagem)) # 6
Cuidado: Tentar acessar um índice inexistente causa erro!

Fixando o Conhecimento:

Dada a string frase = "Aprender Python é divertido":

  1. Qual o tamanho da string?
  2. Acesse o primeiro caractere
  3. Acesse o 10º caractere
  4. Qual o último caractere? (use len)

3. Índice Negativo e Slice de String

Técnicas avançadas para acessar partes da string.

Exercício Prático:

texto = "Desenvolvimento" # Índices negativos (começam do final) print(texto[-1]) # 'o' (último caractere) print(texto[-3]) # 'n' (terceiro a partir do fim) # Slicing [início:fim:passo] print(texto[2:6]) # 'senv' print(texto[:5]) # 'Desen' (do início até índice 5) print(texto[5:]) # 'volvimento' (do índice 5 até fim) print(texto[::2]) # 'Dsvlvmet' (pulando de 2 em 2) print(texto[::-1]) # 'otnemivlovedneseD' (inverte a string)

Fixando o Conhecimento:

Dada frase = "Python é poderosa":

  1. Extraia a palavra "Python"
  2. Extraia a palavra "poderosa" (use índice negativo)
  3. Inverta a string completa
  4. Extraia os caracteres nas posições pares (0, 2, 4...)

4. Operações com String

Operações básicas que podemos realizar com strings.

Exercício Prático:

# Concatenação nome = "Ana" + " " + "Silva" # Repetição risadas = "ha" * 5 # "hahahahaha" # Verificação texto = "Python" print("Py" in texto) # True print("java" in texto) # False # Comparação print("abc" == "ABC") # False (case sensitive) print("abc" < "def") # True (ordem alfabética)

Fixando o Conhecimento:

Crie um programa que:

  1. Concatene seu nome e sobrenome
  2. Crie uma string com 10 asteriscos (*) usando repetição
  3. Verifique se seu nome contém a letra "a"
  4. Compare duas strings e mostre qual vem primeiro no dicionário

5. Métodos de String - Apresentação

Strings possuem muitos métodos úteis embutidos.

Método Descrição Exemplo
upper() Converte para maiúsculas "python".upper() → "PYTHON"
lower() Converte para minúsculas "PYTHON".lower() → "python"
strip() Remove espaços extras " python ".strip() → "python"
split() Divide a string em partes "a,b,c".split(",") → ["a", "b", "c"]
replace() Substitui partes da string "banana".replace("a", "o") → "bonono"

Exercício Prático:

mensagem = " Aprenda Python Hoje! " print(mensagem.strip()) print(mensagem.lower()) print(mensagem.replace("Hoje", "Agora")) print(mensagem.split()) # Divide nos espaços

Fixando o Conhecimento:

Dada frase = "Python é Fácil de Aprender":

  1. Converta para minúsculas
  2. Substitua "Fácil" por "Divertido"
  3. Divida a string em palavras
  4. Verifique se começa com "Py"

6. Formatação de números

Como formatar números em strings de forma elegante.

Exercício Prático:

valor = 1234.5678 # Formatação com f-strings print(f"R$ {valor:.2f}") # R$ 1234.57 print(f"Número: {valor:,.2f}") # Número: 1,234.57 # Formatação com format() print("Percentual: {:.1%}".format(0.753)) # Percentual: 75.3% print("Hexadecimal: {:x}".format(255)) # Hexadecimal: ff

Fixando o Conhecimento:

Crie strings formatadas que:

  1. Exibam o número π (3.1415926535) com 4 casas decimais
  2. Exibam 1000000 como "1,000,000"
  3. Convertam 0.85 para porcentagem com 1 decimal
  4. Exibam o número 42 em hexadecimal

7. Exercícios String - Fixação

Exercícios Finais:

  1. Crie um programa que:
    • Pergunte o nome completo do usuário
    • Mostre o nome em maiúsculas
    • Conte quantas letras tem (sem espaços)
    • Mostre o primeiro nome e quantas letras tem
  2. Faça um programa que:
    • Pergunte uma data no formato dd/mm/aaaa
    • Extraia o dia, mês e ano separadamente
    • Exiba a data no formato "Dia X, Mês Y do ano Z"
  3. Desenvolva um validador de CPF que:
    • Pergunte um CPF no formato XXX.XXX.XXX-XX
    • Verifique se tem 14 caracteres
    • Verifique se os pontos e traço estão nas posições corretas
  4. Crie um gerador de senhas que:
    • Pergunte o nome do site
    • Pegue as 3 primeiras letras (maiúsculas)
    • Pegue os 2 últimos dígitos do ano atual
    • Adicione "!@#" no final
    • Ex: "google" → "GOO23!@#"
Desafio Extra: Crie um tradutor de "P" que:
  1. Pergunte uma frase ao usuário
  2. Substitua todas as vogais por "p" (ou "P")
  3. Ex: "Hello World" → "Hpllp Wprld"
