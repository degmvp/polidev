Documentação Didática: Manipulação de Arquivos Binários

📌 Introdução

Neste exemplo, são utilizados arquivos binários para gravar e ler dados com segurança usando o módulo pickle.
Essa técnica permite armazenar objetos Python diretamente no disco.

📂 Classe Pessoa e Métodos com Pickle

A classe Pessoa possui atributos simples e métodos para salvar e carregar usando arquivos binários.

import pickle

class Pessoa:
    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade

    def __str__(self):
        return f"Nome: {self.nome} - Idade: {self.idade}"

💾 Salvando Objetos em Arquivo Binário

Usamos pickle.dump para armazenar uma lista de objetos no arquivo pessoas.dat.

pessoas = [
    Pessoa("João", 30),
    Pessoa("Maria", 25),
    Pessoa("Carlos", 40)
]

with open("pessoas.dat", "wb") as f:
    pickle.dump(pessoas, f)

print("Arquivo binário salvo com sucesso.")

📥 Lendo o Arquivo Binário

Com pickle.load recuperamos os dados gravados e exibimos os objetos armazenados.

with open("pessoas.dat", "rb") as f:
    pessoas_carregadas = pickle.load(f)

for p in pessoas_carregadas:
    print(p)

✅ Conclusão

O uso de arquivos binários com pickle permite persistir objetos Python de forma prática.
É ideal para armazenar estruturas de dados complexas que precisam ser reabertas em execuções futuras.
