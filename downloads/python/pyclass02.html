Módulo 2 - Variáveis em Python

Aprofundando nosso conhecimento sobre variáveis e seus tipos.

1. Ordem de Execução dos Programas

Python executa o código linha por linha, de cima para baixo.

Exercício Prático:

Observe a ordem de execução:

# Linhas são executadas em ordem a = 5 b = 10 print(a + b) a = 20 # Esta alteração só afeta daqui para frente print(a + b)

Fixando o Conhecimento:

Qual será a saída deste código?

x = 3 y = x * 2 x = 5 print(y)

2. Tipos de Variáveis

Principais tipos em Python: int (inteiro), float (decimal), str (texto), bool (True/False).

Exercício Prático:

# Descubra os tipos com type() idade = 25 preco = 9.99 nome = "Python" ativo = True print(type(idade)) print(type(preco)) print(type(nome)) print(type(ativo))

Fixando o Conhecimento:

Crie variáveis dos seguintes tipos:

  1. Um número inteiro negativo
  2. Um número decimal muito pequeno
  3. Uma string com seu email
  4. Um valor booleano falso
  5. Verifique os tipos com type()

3. Misturando Tipos de Variáveis

Algumas operações entre tipos diferentes podem causar erros.

Exercício Prático:

numero = 10 texto = "5" # Isso funciona? print(numero + int(texto)) # Convertendo texto para número # Isso causa erro? print(str(numero) + texto) # Convertendo número para texto
Atenção: Não podemos somar diretamente um número com texto!

Fixando o Conhecimento:

Corrija os erros nestes códigos:

# Código 1 idade = "25" print(idade + 5) # Código 2 preco = 19.99 print("O preço é: " + preco)

4. Comando Format

O método format() nos permite criar strings com valores de variáveis.

Exercício Prático:

nome = "Maria" idade = 30 altura = 1.65 # Usando format print("{} tem {} anos e {:.2f}m de altura".format(nome, idade, altura)) # f-strings (Python 3.6+) print(f"{nome} tem {idade} anos e {altura:.2f}m de altura")

Fixando o Conhecimento:

Crie uma mensagem que mostre:

  1. Seu nome completo
  2. Sua idade
  3. Seu peso (com 1 casa decimal)
  4. Use tanto format quanto f-string

5. String e Mudança de Tipos de Variáveis

Podemos converter entre tipos usando int(), float(), str(), bool().

Exercício Prático:

# Conversões entre tipos numero_texto = "123" numero_inteiro = int(numero_texto) numero_decimal = float(numero_texto) print(numero_inteiro + 7) print(numero_decimal / 2) # Convertendo para string valor = 45.7 texto_valor = str(valor) print("O valor é " + texto_valor)
