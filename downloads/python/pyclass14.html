Módulo 14 - Integração Python com Arquivos TXT e PDF

Aprenda a manipular arquivos de texto e PDFs com Python.

1. Lendo Arquivos TXT

Operações básicas de leitura com arquivos texto.

Exercício Prático:

# Abrindo e lendo todo o conteúdo arquivo = open('dados.txt', 'r', encoding='utf-8') conteudo = arquivo.read() arquivo.close() print(conteudo) # Lendo linha por linha arquivo = open('dados.txt', 'r', encoding='utf-8') linhas = arquivo.readlines() arquivo.close() for linha in linhas: print(linha.strip()) # strip() remove \n no final # Lendo com loop direto no arquivo arquivo = open('dados.txt', 'r', encoding='utf-8') for linha in arquivo: print(linha.strip()) arquivo.close()
Importante: Sempre especifique o encoding (utf-8 é o mais comum) para evitar problemas com caracteres especiais.

Fixando o Conhecimento:

Crie um programa que:

  1. Conte o número de linhas de um arquivo
  2. Encontre todas as linhas que contêm uma palavra específica
  3. Calcule a média de números em um arquivo (um por linha)

2. Escrevendo em Arquivo TXT

Criação e modificação de arquivos texto.

Exercício Prático:

# Escrevendo em um arquivo novo with open('saida.txt', 'w', encoding='utf-8') as arquivo: arquivo.write("Primeira linha\n") arquivo.write("Segunda linha\n") arquivo.writelines(["Terceira linha\n", "Quarta linha\n"]) # Adicionando a um arquivo existente with open('saida.txt', 'a', encoding='utf-8') as arquivo: arquivo.write("Nova linha adicionada no final\n") # Modos de abertura: # 'r' - leitura (padrão) # 'w' - escrita (sobrescreve) # 'a' - adicionar no final # 'r+' - leitura e escrita

Fixando o Conhecimento:

Crie programas que:

  1. Gere um arquivo com a tabuada de 1 a 10
  2. Combine o conteúdo de dois arquivos em um terceiro
  3. Registre dados de usuário em um arquivo de log

3. Método With

Gerenciamento seguro de recursos com context managers.

Exercício Prático:

# Forma tradicional (precisa fechar manualmente) arquivo = open('dados.txt', 'r') try: conteudo = arquivo.read() finally: arquivo.close() # Com with (fecha automaticamente) with open('dados.txt', 'r') as arquivo: conteudo = arquivo.read() # Arquivo já está fechado aqui # Múltiplos arquivos with open('origem.txt', 'r') as origem, open('destino.txt', 'w') as destino: destino.write(origem.read())
Melhor prática: Sempre use with para garantir que recursos sejam liberados corretamente.

Fixando o Conhecimento:

Reescreva estes exemplos usando with:

# 1. Leitura sem with arq = open('dados.txt', 'r') dados = arq.read() arq.close() # 2. Escrita sem with log = open('log.txt', 'a') log.write("Nova entrada\n") log.close()

4. Integração Python e PDF - Como funciona

Introdução às bibliotecas para manipulação de PDF.

Exercício Prático:

# Instalação das bibliotecas (executar no terminal) # pip install PyPDF2 tabula-py pdf2image # Extraindo texto com PyPDF2 from PyPDF2 import PdfReader reader = PdfReader("documento.pdf") texto = "" for pagina in reader.pages: texto += pagina.extract_text() print(texto) # Contando páginas print(f"Total de páginas: {len(reader.pages)}")
Limitações: Nem todos os PDFs permitem extração perfeita de texto (especialmente PDFs baseados em imagens).

Fixando o Conhecimento:

Crie um programa que:

  1. Conte o número de páginas de um PDF
  2. Extraia todo o texto de um PDF para um arquivo TXT
  3. Verifique se um PDF contém uma palavra específica

5. tabula - Lendo tabelas em PDF

Extraindo dados tabulares de PDFs.

Exercício Prático:

# Extraindo tabelas com tabula-py import tabula # Lendo todas as tabelas tabelas = tabula.read_pdf("dados.pdf", pages='all') # Lendo tabela específica (página 1, área coordenadas) tabela = tabula.read_pdf("dados.pdf", pages=1, area=(100, 50, 400, 500)) # Convertendo para DataFrame pandas df = tabula.read_pdf("dados.pdf", pages=1)[0] # Salvando como CSV tabula.convert_into("dados.pdf", "dados.csv", output_format="csv", pages='all')
Dica: Para tabelas complexas, especifique as coordenadas da área (y1, x1, y2, x2).

Fixando o Conhecimento:

Extraia tabelas de um PDF e:

  1. Salve como arquivo CSV
  2. Calcule estatísticas básicas dos dados
  3. Filtre linhas com base em critérios

6. Extrair imagens de PDF

Recuperando imagens embutidas em documentos PDF.

Exercício Prático:

# Extraindo imagens com pdf2image from pdf2image import convert_from_path # Convertendo PDF para imagens imagens = convert_from_path('documento.pdf', dpi=300) # Salvando imagens for i, img in enumerate(imagens): img.save(f'pagina_{i+1}.jpg', 'JPEG') # Extraindo apenas páginas específicas imagens = convert_from_path('documento.pdf', first_page=2, last_page=5)
Requisitos: pdf2image requer poppler instalado no sistema (no Windows, adicione ao PATH).

Fixando o Conhecimento:

Crie um programa que:

  1. Converta um PDF em imagens JPG
  2. Extraia apenas páginas pares
  3. Redimensione as imagens para 50% do tamanho original
