Documentação Didática: Trabalhando com Múltiplas Instâncias

📌 Introdução

Neste exemplo, exploramos como instanciar múltiplos objetos de uma mesma classe, passando parâmetros diferentes.

🔧 Classe Produto com Parâmetros

A classe Produto é construída com nome, preço e quantidade. Um método exibe os dados formatados.

class Produto:
    def __init__(self, nome, preco, quantidade):
        self.nome = nome
        self.preco = preco
        self.quantidade = quantidade

    def mostrar_produto(self):
        print(f"Produto: {self.nome}")
        print(f"Preço: R$ {self.preco}")
        print(f"Quantidade: {self.quantidade} unidades")

# Criando instâncias diferentes
p1 = Produto("Caneta", 2.50, 10)
p2 = Produto("Caderno", 15.00, 5)
p3 = Produto("Borracha", 1.00, 20)

# Exibindo os dados
p1.mostrar_produto()
print("-" * 40)
p2.mostrar_produto()
print("-" * 40)
p3.mostrar_produto()

✅ Explicação Final

Cada objeto criado é independente, mas compartilha a mesma estrutura da classe.
Este é um passo importante para trabalhar com **coleções de objetos** e **lógicas orientadas a dados**.
