Módulo 5 - Listas em Python

Domine uma das estruturas de dados mais importantes do Python.

1. Listas em Python

Listas são coleções ordenadas e mutáveis de itens.

Exercício Prático:

# Criando listas frutas = ["maçã", "banana", "laranja"] numeros = [1, 2, 3, 4, 5] misturado = [10, "texto", True, 3.14] print(frutas) print(numeros[0]) # Acessando o primeiro elemento
Dica: Listas podem conter qualquer tipo de dado, inclusive outras listas!

Fixando o Conhecimento:

Crie listas para representar:

  1. Seus 3 filmes favoritos
  2. Os primeiros 5 números pares
  3. Dados pessoais (nome, idade, altura)

2. Índices em Lista, Consultando e Modificando Valores

Acesse e modifique elementos usando índices.

Exercício Prático:

linguagens = ["Python", "Java", "C++", "JavaScript"] # Acessando elementos print(linguagens[0]) # Python print(linguagens[-1]) # JavaScript (último elemento) # Modificando elementos linguagens[1] = "Ruby" print(linguagens) # ["Python", "Ruby", "C++", "JavaScript"]
Cuidado: Índices fora do range causam erro!

Fixando o Conhecimento:

Dada a lista cores = ["vermelho", "verde", "azul", "amarelo"]:

  1. Acesse a segunda cor
  2. Mude "azul" para "roxo"
  3. Acesse a última cor (use índice negativo)

3. Descobrindo o Índice de um item na lista

Encontre a posição de elementos com o método index().

Exercício Prático:

frutas = ["maçã", "banana", "laranja", "uva"] # Encontrando o índice print(frutas.index("banana")) # 1 print(frutas.index("uva")) # 3 # Tratando erros quando o item não existe if "manga" in frutas: print(frutas.index("manga")) else: print("Manga não está na lista")

Fixando o Conhecimento:

Crie um programa que:

  1. Pergunte ao usuário uma fruta
  2. Verifique se está na lista frutas = ["maçã", "banana", "laranja"]
  3. Se estiver, mostre sua posição
  4. Se não, informe "Fruta não encontrada"

4. Adicionar e Remover itens da Lista

Métodos para modificar o conteúdo da lista.

Método Descrição Exemplo
append() Adiciona item ao final lista.append(x)
insert() Insere item em posição específica lista.insert(1, x)
remove() Remove primeiro item com valor x lista.remove(x)
pop() Remove e retorna item na posição (ou último) lista.pop(1)
clear() Remove todos os itens lista.clear()

Exercício Prático:

numeros = [1, 2, 3] numeros.append(4) # [1, 2, 3, 4] numeros.insert(1, 5) # [1, 5, 2, 3, 4] numeros.remove(2) # [1, 5, 3, 4] ultimo = numeros.pop() # [1, 5, 3], ultimo=4 numeros.clear() # []

Fixando o Conhecimento:

Crie um programa que gerencie uma lista de compras:

  1. Comece com lista vazia
  2. Permita adicionar itens
  3. Permita remover itens
  4. Mostre a lista atualizada após cada operação

5. Tamanho de Lista, Maior e Menor Valor

Funções úteis para trabalhar com listas numéricas.

Exercício Prático:

valores = [15, 7, 27, 39, 11] print(len(valores)) # Tamanho: 5 print(max(valores)) # Máximo: 39 print(min(valores)) # Mínimo: 7 print(sum(valores)) # Soma: 99
Dica: max() e min() também funcionam com strings, comparando ordem alfabética.

Fixando o Conhecimento:

Dada a lista notas = [7.5, 8.0, 6.5, 9.0, 5.5]:

  1. Calcule a quantidade de notas
  2. Encontre a maior nota
  3. Encontre a menor nota
  4. Calcule a média das notas

6. Juntar Listas e Ordenar

Combine listas e organize seus elementos.

Exercício Prático:

lista1 = [1, 2, 3] lista2 = [4, 5, 6] # Juntando listas combinada = lista1 + lista2 # [1, 2, 3, 4, 5, 6] # Ordenando numeros = [3, 1, 4, 2] numeros.sort() # [1, 2, 3, 4] numeros.sort(reverse=True) # [4, 3, 2, 1] # Ordenação sem modificar ordenada = sorted(numeros) # Nova lista ordenada

Fixando o Conhecimento:

Crie um programa que:

  1. Tenha duas listas de números
  2. Una-as em uma única lista
  3. Ordene em ordem crescente e decrescente
  4. Mostre os três maiores números

7. Print e Join em Listas

Exibindo listas de forma elegante.

Exercício Prático:

frutas = ["maçã", "banana", "laranja"] # Print direto print(frutas) # ['maçã', 'banana', 'laranja'] # Join para criar string print(", ".join(frutas)) # maçã, banana, laranja print("\n".join(frutas)) # Cada fruta em uma linha
Atenção: join() só funciona com strings!

Fixando o Conhecimento:

Dada a lista nomes = ["Ana", "Carlos", "Beatriz"]:

  1. Exiba os nomes separados por hífen
  2. Exiba um nome por linha
  3. Converta a lista para uma string no formato "Nomes: Ana, Carlos e Beatriz"

Métodos adicionais para manipulação de listas.

Exercício Prático:

Fixando o Conhecimento:

Crie um programa que:

  1. Conte quantas vezes aparece o número 5 em [1, 5, 3, 5, 5, 2]
  2. Inverta a ordem de uma lista de palavras
  3. Crie uma cópia segura de uma lista e modifique a cópia

Formas concisas de modificar valores.

11. Copiar e Igualdade de Listas

Entenda a diferença entre cópia superficial e profunda.

Exercício Prático:

# Cópia superficial original = [1, 2, 3] copia = original.copy() original[0] = 99 print(original) # [99, 2, 3] print(copia) # [1, 2, 3] (não foi alterada) # Igualdade a = [1, 2, 3] b = [1, 2, 3] print(a == b) # True (mesmos valores) print(a is b) # False (objetos diferentes)

Fixando o Conhecimento:

Explique a diferença entre:

  1. lista1 = lista2
  2. lista1 = lista2.copy()
  3. lista1 = list(lista2)
