Módulo 15 - Integração Python-SQL

1. pyodbc - Configurações

O pyodbc é uma biblioteca Python que permite conexão com bancos de dados SQL. Para começar:

import pyodbc

# Configuração da conexão
conn = pyodbc.connect(
    Driver='{SQL Server}',  # Ou outro driver como MySQL, PostgreSQL
    Server='nome_do_servidor',
    Database='nome_do_banco',
    UID='usuario',
    PWD='senha'
)

# Criar um cursor para executar comandos SQL
cursor = conn.cursor()
Nota: Os parâmetros de conexão variam conforme o banco de dados utilizado. Consulte a documentação do seu SGBD para os detalhes específicos.

2. Read no Banco de Dados

Para ler dados de uma tabela:

# Executar uma consulta SELECT
cursor.execute("SELECT * FROM tabela")

# Recuperar todos os resultados
rows = cursor.fetchall()

# Iterar sobre os resultados
for row in rows:
    print(row)
    
# Ou buscar um registro por vez
row = cursor.fetchone()
while row:
    print(row)
    row = cursor.fetchone()

3. Read no Banco de Dados com o Pandas

Usando pandas para facilitar a manipulação dos dados:

import pandas as pd

# Ler dados diretamente para um DataFrame
query = "SELECT * FROM tabela"
df = pd.read_sql(query, conn)

# Mostrar os primeiros registros
print(df.head())

# Fazer análises com pandas
print(df.describe())
print(df.groupby('coluna').mean())

4. Update no Banco de Dados

Para atualizar registros:

# Atualizar um registro
cursor.execute("UPDATE tabela SET coluna1 = ? WHERE id = ?", ('novo_valor', 1))

# Confirmar a transação
conn.commit()

# Verificar quantas linhas foram afetadas
print(f"Linhas afetadas: {cursor.rowcount}")
Importante: Sempre lembre de executar conn.commit() após operações que modificam o banco de dados (INSERT, UPDATE, DELETE).

5. Delete no Banco de Dados

Para deletar registros:

# Deletar um registro
cursor.execute("DELETE FROM tabela WHERE id = ?", (1,))

# Confirmar a transação
conn.commit()

# Verificar quantas linhas foram afetadas
print(f"Linhas deletadas: {cursor.rowcount}")

6. CRUD no Python - Integração Python e MySQL

Exemplo completo de CRUD (Create, Read, Update, Delete) com MySQL:

import pyodbc
import pandas as pd

# Configuração da conexão MySQL
conn = pyodbc.connect(
    Driver='{MySQL ODBC 8.0 Unicode Driver}',
    Server='localhost',
    Database='meu_banco',
    UID='root',
    PWD='senha'
)
cursor = conn.cursor()

# CREATE (Inserir)
cursor.execute("INSERT INTO clientes (nome, email) VALUES (?, ?)", 
               ('João Silva', 'joao@email.com'))
conn.commit()

# READ (Ler)
cursor.execute("SELECT * FROM clientes")
df = pd.read_sql("SELECT * FROM clientes", conn)
print("Clientes:")
print(df)

# UPDATE (Atualizar)
cursor.execute("UPDATE clientes SET email = ? WHERE nome = ?", 
               ('joao.novo@email.com', 'João Silva'))
conn.commit()

# DELETE (Remover)
cursor.execute("DELETE FROM clientes WHERE nome = ?", ('João Silva',))
conn.commit()

# Fechar conexão
cursor.close()
conn.close()

Dicas importantes:

  • Sempre use parâmetros (?) para evitar SQL injection
  • Não esqueça de fazer commit() após INSERT, UPDATE ou DELETE
  • Feche a conexão quando não for mais necessária
  • Para operações em lote, considere usar executemany()

Exemplo de executemany:

dados = [
    ('Maria', 'maria@email.com'),
    ('Pedro', 'pedro@email.com'),
    ('Ana', 'ana@email.com')
]
cursor.executemany("INSERT INTO clientes (nome, email) VALUES (?, ?)", dados)
conn.commit()
