Módulo 7 - Loops While em Python

Aprenda a criar loops baseados em condições com a estrutura while.

1. Estrutura While

O loop while repete um bloco enquanto uma condição for verdadeira.

Exercício Prático:

# Contagem regressiva contador = 5 while contador > 0: print(contador) contador -= 1 print("Decolar!") # Validação de entrada resposta = '' while resposta not in ['sim', 'não']: resposta = input("Você quer continuar? (sim/não): ").lower()
Dica: Use while quando não souber quantas iterações serão necessárias.

Fixando o Conhecimento:

Crie programas que:

  1. Some números digitados até que o usuário digite 0
  2. Gere a sequência de Collatz para um número inicial
  3. Peça uma senha até que o usuário digite a correta

2. Loop Infinito no While

Cuidado com loops infinitos - sempre tenha uma condição de saída.

Exercício Prático:

# Loop infinito intencional com break while True: comando = input("Digite 'sair' para terminar: ") if comando.lower() == 'sair': break print("Executando comando...") # Loop infinito acidental (cuidado!) # contador = 5 # while contador > 0: # print(contador) # # Esqueceu de decrementar contador!
Cuidado: Sempre teste seus loops while para garantir que terminam!

Fixando o Conhecimento:

Corrija estes loops infinitos:

# 1. Loop que não termina x = 10 while x > 0: print(x) x += 1 # Deveria ser -= 1 # 2. Validação problemática numero = -1 while numero < 0: numero = int(input("Digite um número positivo: "))

3. Exercício de While Fixação

Exercícios Finais:

  1. Implemente uma calculadora que:
    • Permita fazer várias operações
    • Pergunte se quer continuar após cada cálculo
    • Mostre histórico de operações ao sair
  2. Crie um jogo de adivinhação onde:
    • O computador sorteia um número
    • O usuário tenta adivinhar
    • O programa dá dicas (maior/menor)
    • Conta o número de tentativas
  3. Desenvolva um sistema de login que:
    • Permita apenas 3 tentativas
    • Bloqueie após tentativas esgotadas
    • Mostre mensagens adequadas
