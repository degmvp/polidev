Mix-Adv 33 - Listas em Python

O que são listas?

Listas são estruturas de dados que armazenam múltiplos elementos em uma única variável, permitindo modificar, adicionar e remover itens.

Como criar uma lista

frutas = ["maçã", "banana", "laranja"]
numeros = [1, 2, 3, 4, 5]
lista_mista = [1, "texto", True, 3.14]

Acessando elementos

print(frutas[0])  # maçã
print(numeros[-1])  # 5 (último elemento)

Adicionando elementos

frutas.append("uva")
frutas.insert(1, "morango")

Removendo elementos

frutas.remove("banana")  # remove pela primeira ocorrência
del frutas[0]  # remove pelo índice
frutas.pop()  # remove o último elemento

Iterando sobre listas

for fruta in frutas:
    print(fruta)

Funções úteis

  • len(lista) — retorna o tamanho da lista.
  • min(lista) e max(lista) — menor e maior elemento.
  • sorted(lista) — retorna a lista ordenada.

Resumo

Listas são versáteis e fundamentais para manipulação de coleções em Python.
