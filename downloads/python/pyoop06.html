Módulo 5 - Polimorfismo

O que é Polimorfismo?

Polimorfismo é a capacidade de diferentes classes responderem a uma mesma mensagem (método) de maneiras diferentes.

É uma forma de reaproveitar nomes de métodos mantendo comportamentos distintos para cada classe.

Tipos de Polimorfismo

  • Polimorfismo por sobrescrita (override): Subclasses reimplementam métodos da superclasse.
  • Polimorfismo por duck typing: "Se anda como um pato e grasna como um pato, então é um pato".

Exemplo com sobrescrita:

class Forma:
def desenhar(self):
print("Desenhando forma genérica")

class Circulo(Forma):
def desenhar(self):
print("Desenhando círculo")

class Quadrado(Forma):
def desenhar(self):
print("Desenhando quadrado")

# Função que usa polimorfismo
def renderizar_forma(forma):
forma.desenhar()

# Teste
formas = [Circulo(), Quadrado(), Forma()]
for f in formas:
renderizar_forma(f)

Exemplo com duck typing:

Não importa o tipo do objeto, mas sim se ele possui o método esperado.

class Pato:
def fazer_som(self):
print("Quack!")

class Cachorro:
def fazer_som(self):
print("Au au!")

def emitir_som(animal):
animal.fazer_som()

emitir_som(Pato()) # Quack!
emitir_som(Cachorro()) # Au au!

Resumo

O polimorfismo permite escrever código mais flexível, reutilizável e desacoplado. Ele é essencial para trabalhar com coleções de objetos diversos que compartilham comportamentos semelhantes.
