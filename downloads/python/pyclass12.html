1. O que são Módulos e qual a importância

Módulos são arquivos Python que contêm definições e instruções reutilizáveis.

Exercício Prático:

# Importando um módulo completo import math # Usando funções do módulo print(math.sqrt(25)) # 5.0 print(math.pi) # 3.141592653589793 # Importando com alias import math as m print(m.cos(m.pi)) # -1.0 # Importando itens específicos from math import pow, factorial print(pow(2, 3)) # 8.0 print(factorial(5)) # 120
Vantagens: Organização, reuso de código, namespaces separados e colaboração.

Fixando o Conhecimento:

Crie um arquivo meu_modulo.py com:

  1. Uma função que calcula a área de um círculo
  2. Uma constante com o valor de gravidade (9.8)
  3. Importe e use esses itens em outro arquivo

3. Biblioteca time - Trabalhando com Datas no Python

Operações básicas com tempo usando o módulo time.

Exercício Prático:

import time # Tempo atual em segundos desde epoch agora = time.time() print(agora) # Tempo local como struct_time local = time.localtime() print(local) print(f"Ano: {local.tm_year}, Mês: {local.tm_mon}") # Formatação de tempo formatado = time.strftime("%d/%m/%Y %H:%M:%S", local) print(formatado) # Dormir por 2 segundos print("Início") time.sleep(2) print("Fim")

Fixando o Conhecimento:

Crie um programa que:

  1. Mostre a data e hora atual formatada
  2. Calcule quanto tempo leva para executar um loop
  3. Converta um timestamp específico para formato legível

4. Métodos Avançados da biblioteca time

Funções mais sofisticadas para manipulação de tempo.

Exercício Prático:

import time # Medindo tempo de execução inicio = time.perf_counter() # Código a ser medido for _ in range(1000000): pass fim = time.perf_counter() print(f"Tempo: {fim - inicio:.6f} segundos") # Process time (só tempo de CPU) inicio_cpu = time.process_time() # ... código ... fim_cpu = time.process_time() # Conversão de string para struct_time data_str = "25/12/2023" struct_data = time.strptime(data_str, "%d/%m/%Y") print(struct_data)

Fixando o Conhecimento:

Implemente um cronômetro que:

  3. Mostre o tempo decorrido com precisão

6. Biblioteca datetime - Métodos Avançados

Manipulação mais intuitiva de datas e horas.

Exercício Prático:

from datetime import datetime, timedelta # Data e hora atual agora = datetime.now() print(agora) print(agora.year, agora.month, agora.day) # Operações com datas amanha = agora + timedelta(days=1) print(amanha) # Formatação print(agora.strftime("%A, %d de %B de %Y")) # Criação de data específica natal = datetime(2023, 12, 25) print((natal - agora).days, "dias para o Natal")

Fixando o Conhecimento:

  3. Mostre quantos dias faltam para o próximo aniversário

8. Módulo de Gráficos - Exibindo Gráficos no Python

Introdução à visualização de dados com Matplotlib.

Exercício Prático:

import matplotlib.pyplot as plt # Dados x = [1, 2, 3, 4, 5] y = [2, 4, 6, 8, 10] # Gráfico básico plt.plot(x, y) plt.title("Gráfico Linear") plt.xlabel("Eixo X") plt.ylabel("Eixo Y") plt.grid(True) plt.show() # Gráfico de barras nomes = ["A", "B", "C"] valores = [15, 30, 45] plt.bar(nomes, valores) plt.show()

  2. Comparem vendas por mês usando barras

9. Mais edições de Gráficos e Módulo Numpy

Personalização avançada e computação numérica.

Exercício Prático:

import numpy as np import matplotlib.pyplot as plt # Criando arrays com numpy x = np.linspace(0, 10, 100) y = np.sin(x) # Gráfico personalizado plt.figure(figsize=(10, 5)) plt.plot(x, y, 'r--', label='Seno(x)') plt.title("Função Seno", fontsize=16) plt.legend() plt.savefig('grafico.png') # Salvando a figura # Operações com numpy a = np.array([1, 2, 3]) b = np.array([4, 5, 6]) print(a + b) # [5 7 9] print(a * 2) # [2 4 6]

Fixando o Conhecimento:

Combine numpy e matplotlib para:

  1. Plotar múltiplas funções matemáticas em um mesmo gráfico
  2. Criar um gráfico 3D simples
  3. Gerar e visualizar dados aleatórios com distribuição normal
