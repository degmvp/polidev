Mix-Adv 25 - Operadores Lógicos em Python

Operadores Lógicos

Os operadores lógicos em Python permitem combinar expressões booleanas. Os principais são:

  • and: retorna True se ambas as expressões forem verdadeiras.
  • or: retorna True se pelo menos uma das expressões for verdadeira.
  • not: inverte o valor lógico da expressão.

Exemplos de Uso

a = True
b = False

print(a and b)   # False
print(a or b)    # True
print(not a)     # False

Com Condições

Os operadores lógicos são muito utilizados em estruturas condicionais: 

idade = 20
tem_carteira = True

if idade >= 18 and tem_carteira:
    print("Pode dirigir.")
else:
    print("Não pode dirigir.")

Curiosidades

Python avalia expressões da esquerda para a direita e para nos primeiros valores que tornam a expressão completa (curto-circuito lógico).
